RELATED TOPICS

View list of registered customers

 

 

The airline staff can use the Maintain Customer screen.

 

Permanent registration of a customer is the process by which a customer is registered permanently in the system by allotting a permanent customer code to the customer.

 

You can view the details of a registered customer on the basis of customer code which is auto generated by the system when a permanent registration is carried out. The permanent registration of a customer can be either carried out for a new customer or for an already existing customer, who was registered temporarily.

 

When you access the screen, default values are displayed in the following fields:

 

·          Station – the default login station

·          Status – ‘Active’

·          Default Notify Value – ‘Email’

 

Functions

Register a customer

View details of a registered customer

Modify details of a registered customer

 

Note:

To view/modify customer details:

·        If you are not aware of the Cust Code of the customer, you can use the List Customers screen. In this screen, provide your search criteria, locate the required customer and then view/modify the details.

Locate a customer and view/modify details

·        However, if you know the Cust Code of the customer, you can use the Maintain Customer screen. Specify the Cust Code of the customer, and then display the details for viewing/modification.

 

Register a customer

 

1.      Open the Maintain Customer screen ( Airline Functions > Masters > Customer > Maintain Customer).

 

2.      In this screen, type the Cust Code. (optional)

 

3.      Click List. (mandatory)

The customer details corresponding to the search criteria are displayed.

If the specified customer:

•        exists in the system, the details of the customer are displayed.

•        does not exist, a message appears enquiring if you wish to create a new customer.  Click Yes to proceed. The buttons and links on the screen become enabled.

Note:

If Cust Code is not specified, the system generates a Cust Code for the particular customer after you enter the customer details and click Save.

 

4.      In thisscreen, specify the customer details. (mandatory)

  • Type the Customer Name.
  • Type or selectthe Station.
  • Select the Customer Type.

·          Type the City.

·          Type or select the Country.

Note:

If the Customer Type selected is ‘Agent’, then type or select the Export IATA Agent Code. This is mandatory.

5.      Under the Customer Details tab, specify the details. (optional)

·          Specify at least one primary contact.

·          The contact details corresponding to the Default notify value which was selected from the drop down list should be mentioned in its respective field in the table. For example, if the value selected in the Default notify value drop down is E-Mail, then the E-Mail field of the key contact becomes mandatory

Note:

·          You can add more than one key contact details. For this, click the Add link and specify the details in the new row that appears. However, only one key contact can be specified for a particular department.

·          After permanent registration of the customer is over all the transactions which were carried out using the temporary ID will be transferred to the permanently registered customer by the system .

 

Note:
Status appears as ‘ACTIVE’ , by default, and this cannot be edited.

 

6.      If required, you can specify agent details in the Agent tab. (optional). For this:

  • Click the Add link.
  • Specify the details in the row that gets added.

 

Note:
If agent details are being furnished, then:

·                         It is mandatory to specify Station and Agent Code.

·          Station is to be specified first; the agents based on the specified station are then listed in the Agent LOV.

·                         For a station, agent codes cannot be repeated.

  • To add more agent details click the Add link and specify the details in the row that gets added. To delete an agent, select a row and click the Delete link.

 

7.      Click Save. (mandatory)

The customer details are saved with Status ‘ACTIVE’. A customer code is generated if a customer code was not specified initially.

 

8.      Click OK in the message box that appears. (mandatory)

 

9.      Click Close to exit the screen.
