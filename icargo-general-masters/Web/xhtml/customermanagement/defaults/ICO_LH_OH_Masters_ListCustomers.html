Register a customer

Airline Functions > Masters > Customer > List Customers.


Frequent customers of an airline are registered permanently in the system and they are provided with a permanent customer code or ID. This customer code is generated by the system at the time of customer registration. For all further transactions as well as identification of the customer, the respective customer code is used.

 

The Airline Staff can use the List Custome rs screen.

 

The screen provides links to Maintain Customer screen.

To view/modify customer details:

·        If you are not aware of the Cust Code of the customer, you can use the List Customer screen. In this screen, provide your search criteria, locate the required customer and then view/modify its details.

 

·        However, if you know the Cust Code of the customer, you can use the Customer Registration screen. Specify the Cust Code of the customer, and then display its details for viewing/modification.

View list of customers

 

1.      Open the List Customers screen ( Airline Functions > Masters > Customer > List Customers).

 

2.      In the List Customer screen, specify the search criteria. None are mandatory:

 

3.      Click List. (mandatory)

The list of customers that satisfy the search criteria appear in the Customer Details table.

Note:

To view all existing customers, click List without specifying any search criteria.

 

4.      To exit the screen, click Close.

Field/Button Description – List Customers screen

 

Locate a customer and view/modify details

If you are not aware of the Cust Code of the customer, you can use the List Customer screen to search for, locate and then view/modify details of the customer.

 

Note:

However, if you know the Cust Code of the customer, go to Customer Registration screen. Specify the Cust Code and then display customer details for viewing/modification.

View details of an existing customer

Modify details of an existing customer

 

1.      Open the List Customers screen ( Airline Functions > Masters > Customer > List Customers).

 

2.      In the List Customer screen, list the customers. (mandatory)

 

3.      In the Customer Details table, select the required row. (mandatory)

 

4.      Click Details. (mandatory)

The Customer Registration screen appears with the details of the customer.

 

5.      In the Customer Registration screen:

 

a)    Make the required changes. All fields except Customer Code are editable.

Modify details of an existing customer

b)    Click Save.

The modifications are saved.

c)    Click Close.

The List Customer screen reappears with the updated customer details.

Field/Button Description – Customer Registration screen

 

6.      To exit the screen, click Close.

Field/Button Description – List Customers screen

 

Activate a customer

 

1.      Open the List Customers screen ( Airline Functions > Masters > Customer > List Customers).

 

2.      In the List Customer screen, list the customers. (mandatory)

 

3.      In the Customer Details table select the row to be activated. (mandatory)

Note:

·        You can activate multiple customers at a time. To do this, select all the corresponding rows.

·        Only customers with status ‘Inactive’ can be activated.

 

4.      Click Activate. (mandatory)

Click Yes in the confirmation message that appears.

The selected customers are activated.

 

5.      To exit the screen, click Close.

Field/Button Description – List Customers screen

 

Inactivate a customer

 

1.      Open the List Customers screen ( Airline Functions > Masters > Customer > List Customers).

 

2.      In the List Customer screen, list the customers. (mandatory)

 

3.      In the Customer Details table select the row to be inactivated. (mandatory)

Note:

·        You can inactivate multiple customers at a time. To do this, select all the corresponding rows.

·        Only customers with status ‘Active’ can be inactivated.

 

4.      Click Inactivate. (mandatory)

Click Yes in the confirmation message that appears.

The Reason screen appears.

 

5.      In the Reason screen:

a)    Type the Remarks. (mandatory)

Note:

To discontinue the process and return to the List Customer screen, click Cancel.

b)    Click OK. (mandatory)

The List Customer screen reappears.

The selected customers are inactivated.

Field/Button Description – Reason screen

 

6.      To exit the screen, click Close.

Field/Button Description – List Customers screen

 

Blacklist a customer

 

1.      Open the List Customers screen ( Airline Functions > Masters > Customer > List Customers).

 

2.      In the List Customer screen, list the customers. (mandatory)

 

3.      In the Customer Details table, select the row to be blacklisted. (mandatory)

Note:

·        You can black list multiple customers at a time. To do this, select all the corresponding rows.

·        Only customers with status ‘Active’ can be blacklisted.

 

4.      Click Black List. (mandatory)

Click Yes in the confirmation message that appears.

The Reason screen appears.

 

5.      In the Reason screen:

a)   Type the remarks. (mandatory)

Note:

To discontinue the process and return to the List Customer screen, click Cancel.

b)    Click OK. (mandatory)

The List Customer screen reappears.

The selected customers are blacklisted.

Field/Button Description – Reason screen

 

6.      To exit the screen, click Close.

Field/Button Description – List Customers screen

 

Revoke a customer from blacklist

1.      Open the List Customers screen ( Airline Functions > Masters > Customer > List Customers).

 

2.      In the List Customer screen, list the customers. (mandatory)

 

3.      In the Customer Details table select the row to be revoked from blacklist. (mandatory)

Note:

·        You can revoke multiple customers at a time from the blacklist. To do this, select all the corresponding rows.

·        Only customers with status ‘Blacklisted’ can be revoked from the blacklist.

 

4.      Click Revoke Black List. (mandatory)

Click Yes in the confirmation message that appears.

The selected customers are revoked from the blacklist.

 

5.      To exit the screen, click Close.

Field/Button Description – List Customers screen

Upload TSA lists

 

1.      Open the List Customers screen ( Airline Functions > Masters > Customer > List Customers).

 

2.      In the List Customers screen, click Upload TSA Lists. (mandatory)

The Load From File screen appears.

 

3.      In the Load From File screen,

a)       Select the File Type

b)       Type or browse for the File Name

c)        Click Load

The TSA file is loaded and the List Customers screen reappears

 

4.      To exit the screen, click Close.
