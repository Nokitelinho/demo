To access the List Agreement screen:
Select the ULD Management tab. On the Transaction Management menu, click List Agreement.

RELATED TOPICS

Create a ULD agreement

 

When you access the screen, default values are displayed in the following fields:

·        Party Type – ‘All'

·        Txn Type(Transaction Type) - ‘All’

·        Agreement Status – ‘All’

 

The screen provides a link to the Maintain ULD Agreement screen.

 

The System Administrator can use the List Agreement screen.

Note:

 

·        To create a new ULD agreement, click Create in the List Agreement screen. This takes you to the Maintain ULD Agreement screen, where you can create a ULD agreement.

 

·        To view/modify ULD agreement details:

-   If you are not aware of the agreement number of the ULD agreement, you can use the List Agreement screen. In this screen, provide your search criteria, locate the required ULD agreement and then view/modify its details.

 

-   However, if you know the agreement number of the ULD agreement, you can use the Maintain ULD Agreement screen. Specify the agreement number of the booking, and then display its details for viewing/modification.

View list of ULD agreement details

 

1.      Select the ULD Management tab. On the Transaction Management menu, click List Agreement to get the List Agreement screen.

 

2.      In the List Agreement screen, specify the search criteria. None are mandatory .

 

3.      Click List. (mandatory)

The list of ULD agreements that satisfy the specified search criteria appear in the Agreement Details table.

Note:

To view all existing ULD agreements in the system, click List without specifying any search criteria.

 

4.      To exit the screen, click Close.

Locate a ULD agreement and view/modify details

If you are not aware of the agreement number of the ULD agreement, you can use the List Agreement screen. In this screen, provide your search criteria, locate the required ULD agreement and then view/modify its details.

 

Note:

However, if you know the agreement number of the ULD agreement, you can use the Maintain ULD Agreement screen. Specify the agreement number of the booking, and then display its details for viewing/modification.

View the details of an existing ULD agreement

Modify details of an existing ULD agreement

 

1.      Select the ULD Management tab. On the Transaction Management menu, click List Agreement to get the List Agreement screen.

 

2.      In the Maintain Flight Schedule screen, list the ULD agreements. (mandatory)

 

3.      In the Agreement Details table, select the required row. (mandatory)

 

4.      Click View. (mandatory)

The Maintain ULD Agreement screen appears with the details of the ULD agreement.

 

5.      In the Maintain ULD Agreement screen.

a)    Make the required changes. All the fields except Agreement Number and Agreement Status are editable.

Modify details of an existing ULD agreement

b)    Click Save. (mandatory)

The modifications are saved.

c)    Click Close. (mandatory)

The List Agreement screen reappears. The updated ULD agreement is displayed in the Agreement Details table.

Field/Button Description – Maintain ULD Agreement screen

 

6.      To exit the screen, click Close.

Field/Button Description – List Agreement screen

 

Activate a ULD agreement

 

1.      Select the ULD Management tab. On the Transaction Management menu, click List Agreement to get the List Agreement screen.

 

2.      In the Maintain Flight Schedule screen, list the ULD agreements. (mandatory)

 

3.      In the Agreement Details table, select the required row. (mandatory)

Note:

·        You can activate multiple agreements at a time. To do this, select all the corresponding rows.

·        Only agreements with status ‘Inactive’ can be activated.

 

4.      Click Activate. (mandatory)

Click Yes in the confirmation message that appears.

The selected agreements areactivated.

 

5.      To exit the screen, click Close.

Field/Button Description – List Agreement screen

 

Inactivate a ULD agreement

 

1.      Select the ULD Management tab. On the Transaction Management menu, click List Agreement to get the List Agreement screen.

 

2.      In the Maintain Flight Schedule screen, list the ULD agreements. (mandatory)

 

3.      In the Agreement Details table, select the required row. (mandatory)

Note:

·        You can inactivate multiple agreements at a time. To do this, select all the corresponding rows.

·        Only agreements with status ‘Active’ can be inactivated.

 

4.      Click Inactivate. (mandatory)

Click Yes in the confirmation message that appears.

The selected agreements are inactivated.

 

5.      To exit the screen, click Close.

Field/Button Description – List Agreement screen

 

Print a ULD agreement

 

1.      Select the ULD Management tab. On the Transaction Management menu, click List Agreement to get the List Agreement screen.

 

2.      In the Maintain Flight Schedule screen, list the ULD agreements. (mandatory)

 

3.      Click Print. (mandatory)

The agreement report is printed.

 

4.      To exit the screen, click Close.

Field/Button Description – List Agreement screen

 

Delete a ULD agreement from the system

 

1.      Select the ULD Management tab. On the Transaction Management menu, click List Agreement to get the List Agreement screen.

 

2.      In the Maintain Flight Schedule screen, list the ULD agreements. (mandatory)

 

3.      In the Agreement Details table, select the required row. (mandatory)

Note:

·        You can delete multiple agreements at a time. To do this, select all the corresponding rows.

·        You can only delete inactivated documents.

 

4.      Click Delete. (mandatory)

Click Yes in the confirmation message that appears.

The agreements are deleted from the system.

 

5.      To exit the screen, click Close.
