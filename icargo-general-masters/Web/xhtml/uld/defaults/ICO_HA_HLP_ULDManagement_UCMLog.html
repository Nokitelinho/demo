UCM LOG

(Screen ID – ULD022)

Click here to view screen access information.

To access the UCM Log screen:
Select the ULD Management tab. On the ULD Messages menu, click Reconcile UCM Error logs.

RELATED TOPICS

When you access the screen, default values are displayed in the following fields:

·        Airport – the logged in airport

·        Message Type – ‘ OUT’

·        Message Status – ‘ ALL’

 

The screen provides link to ULD Error Log screen.

 

The following users can use the UCM Log screen.

·        ULD Controller

·        Ground handling agents

View list of UCM OUT/UCM IN errors

 

1.      Select the ULD Management tab. On the ULD Messages menu, click Reconcile UCM Error logs to get the UCM Log screen.

 

2.      In the UCM Log screen: In the Search Criteria section, specify the search criteria. None are mandatory .

Note:

·          To view UCM OUT errors, select the Message Type as ‘OUT’.

·          To view UCM IN errors, select the Message Type as ‘IN’.

 

3.      Click List. (mandatory)

The list of UCM OUT/UCM IN errors matching the search criteria appear in the table.

 

4.      To exit the screen, click Close.

Field/Button Description – UCM Log screen

Reconcile UCM errors for Duplicate UCM OUT messages

 

1.      Select the ULD Management tab. On the ULD Messages menu, click Reconcile UCM Error logs to get the UCM Log screen.

 

2.      In the UCM Log screen: In the Search Criteria section, select the Message Type as ‘OUT’. (mandatory)

 

3.      Click List. (mandatory)

The list of UCM OUT errors that satisfy the specified search criteria appear in the table.

 

4.      In the table, select any two rows corresponding to the same flight for which duplicate UCM OUT messages have to be reconciled. (mandatory)

 

5.      Click Reconcile. (mandatory)

The Duplicate OUT screen appears with the Flight Details and UCM Details.

Note:

UCM errors in Duplicate OUT screen can be reconciled only when a maximum of two rows are taken for reconciliation

 

6.      In the Duplicate OUT screen:

a)     Select the ULDs that should go into the final UCM message (mandatory)

Note:

To discontinue the process and return to the UCM Log screen, click Close.

 

b)     Click OK. (mandatory)

The UCM Log screen reappears with the final UCM message and a new UCM number.

Field/Button Description – Duplicate OUT screen

 

7.      To exit the screen, click Close.

Field/Button Description – UCM Log screen

Reconcile UCM errors for UCM OUT-UCM IN mismatch

For every UCM OUT message sent a corresponding UCM IN message must also be sent from the destination airport. Suppose, the destination airport sends a UCM IN message for incorrect number of ULDs or for ULDs that are not mentioned in the UCM OUT message, then a UCM Mismatch error occurs.

 

1.      Select the ULD Management tab. On the ULD Messages menu, click Reconcile UCM Error logs to get the UCM Log screen.

 

2.      In the UCM Log screen: In the Search Criteria section, select the Message Type as ‘IN’. (mandatory)

 

3.      Click List (mandatory

The list of UCM IN errors that satisfy the specified search criteria appear in the table.

 

4.      In the table, select the error for which UCM mismatch has to be reconciled and click Reconcile. (mandatory)

The MisMatch screen appears

 

5.      In the MisMatch screen:

a)     Select the ULDs that should go into the final UCM message. (mandatory)

Note:

To discontinue the process and return to the UCM Log screen, click Close.

 

b)     Click OK . (mandatory)

The UCM Log screen reappears with the final UCM message and a new UCM No and Message Status changes to ‘Send’ .

Field/Button Description – MisMatch screen

 

6.      To exit the screen, click Close .

Field/Button Description – UCM Log screen

Send UCM

 

1.      Select the ULD Management tab. On the ULD Messages menu, click Reconcile UCM Error logs to get the UCM Log screen.

 

2.      In the UCM Log screen: In the Search Criteria section, specify the search criteria. None are mandatory.

Note:

·          To view UCM OUT errors, select the Message Type as ‘OUT’.

·          To view UCM IN errors, select the Message Type as ‘IN’.

 

3.      Click List. (mandatory)

The list of UCM IN/UCM OUT errors that satisfy the specified search criteria appear in the table.

 

4.      In the table, select the UCM that is to be sent. (mandatory)

 

5.   Click Send UCM (mandatory)

The selected UCM is sent and the message status changes from ‘Pending’ to ‘Send’.

Note:

UCM messages can be sent forcefully, even if there are errors in the UCM messages and ULD details. If an already sent UCM is sent again, its status changes to ‘Re-Send’.

 

5.      To exit the screen, click Close.

Field/Button Description – UCM Log screen
