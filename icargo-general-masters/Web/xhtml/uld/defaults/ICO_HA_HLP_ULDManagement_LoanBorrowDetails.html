LOAN/BORROW DETAILS

(Screen ID – ULD010)

Click here to view screen access information.

To access the Loan/Borrow Details screen:
Select the ULD Management tab. On the Transaction Management menu, click Loan/Borrow Details.

RELATED TOPICS

Create a ULD agreement

 

When you access the screen, default values are displayed in the following fields:

·        Txn Type. – ‘Loan’

·        Txn Nature - ‘Temporary’

·        Txn Airport – the logged in airport

·        Txn Date - the current date

·        Party Type – ‘ Airline ’

 

The screen provides links to Maintain ULD Agreement and Maintain Damage Report screens.

 

The ULD Controller can use the Loan/Borrow Details screen.

Capture loan details of a ULD

 

1.      Select the ULD Management tab. On the Transaction Management menu, click Loan/Borrow Details to get the Loan/Borrow Details screen.

 

2.      In the Loan/Borrow Details screen, select the Txn Type (Transaction Type) as ‘Loan’. (mandatory)

 

3.      In the Txn Details (Transaction Details) section, specify the loan transaction details. The following are mandatory:

·        Select the Txn Nature (Transaction Nature)

·        Select the Party Type

 

4.      In the Party Details section: In the From Party, To Party sections, type or select the Code. (mandatory)

 

5.      In the ULD Detailssection, click the Add link. (mandatory)

The Add Transaction screen appears.

 

6.      In the Add Transaction screen, specify the required ULD details. Type or select the ULD No. This is mandatory.

 

Note:

·        The Ctrl Rcpt No (Control Receipt Number) is generated by the system for ULDs in the system.

·        Only ULDs (ULD numbers) present in the stock can be specified for creating a loan transaction.

·        You can specify the Destn Airport (Destination Airport) of the ULD, if required. If nothing is specified, the logged in airport will be taken as the destination airport

 

7.      You can add more ULD details. To do this, click the Add link and repeat the previous step. ( optional )

 

Note:

·          To delete ULD details, select the required row from the ULD Details table and click the Delete link.

·          You can delete multiple ULDs at a time. To do this, select all the corresponding rows.

 

8.      Click Save. (mandatory)

a)    A message appears enquiring if you wish to save the transaction. Click Yes to proceed.

b)    A message appears enquiring if you wish to generate a UCR report. Click Yes to proceed.

c)    Click OK in the information message that appears. The loan transaction details of the ULD are saved.

 

Note:

In the ULD Details section, if the Condition of any of the ULDs added is selected as ‘DAM’ (Damaged), and if the damage report is not captured for that ULD, then a message appears enquiring if you wish to generate the damage report. Click Yes to generate the damage report. The Maintain Damage Report screen appears with the ULD damage details, if any. You can capture and generate the damage report here.

9.      To exit the screen, click Close.

Capture borrow details (of a ULD/an accessory/both)

 

1.      Select the ULD Management tab. On the Transaction Management menu, click Loan/Borrow Details to get the Loan/Borrow Details screen.

 

2.      In the Loan/Borrow Details screen, select the Txn Type (Transaction Type) as ‘Borrow’. (mandatory)

 

Note:

For a borrow transaction, the default airline Code and Name appears in the To Party section.

 

3.      In the Txn Details (Transaction Details) section, specify the borrow transaction details. The following are mandatory:

·        Select the Txn Nature (Transaction Nature)

·        Select the Party Type (Transaction Nature)

 

4.      In the Party Details section: In the From Party, To Party sections, type or select the Code. (mandatory)

 

5.      In the ULD Details section, click the Add link. (mandatory)

The Add Transaction screen appears.

 

6.      In the Add Transaction screen, specify the required ULD details. Type or select the ULD No. This is mandatory.

 

Note:

·        For a borrow transaction, ULDs that are not present in the stock are to be specified.

·        You can specify the Destn Airport(Destination Airport) of the ULD, if required. If nothing is specified, the logged in airport will be taken as the destination airport

 

7.      You can add more ULD details. To do this, click the Add link and repeat the previous step. ( optional )

 

Note:

·          To delete ULD details, select the required row from the ULD Details table and click the Delete link.

·          You can delete multiple ULDs at a time. To do this, select all the corresponding rows.

 

8.      Click Save. (mandatory)

a)    A message appears enquiring if you wish to save the transaction. Click Yes to proceed.

b)    Click OK in the information message that appears. The borrow transaction details of the ULD are saved.

 

Note:

In the ULD Details section, if the Condition of any of the ULDs added is selected as ‘DAM’ (Damaged), and if the damage report is not captured for that ULD, then a message appears enquiring if you wish to generate the damage report. Click Yes to generate the damage report. The Maintain Damage Report screen appears with the ULD damage details, if any. You can capture and generate the damage report here.

field

The airport at which the transaction takes place.

The condition that determines the demurrage for the ULD.

·   DAM (damaged) - demurrage charged after 6 days
