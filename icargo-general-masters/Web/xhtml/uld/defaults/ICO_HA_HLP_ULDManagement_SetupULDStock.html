To access the Setup ULD Stock screen:
Select the ULD Management tab. On the Stock Management menu click Setup Uld Stock.

 

When you access the screen:

·        If you login with a GHA privilege, the login airport is displayed in the Station field.

·        If you login with an airline privilege, the own airline code is displayed in the Airline Code field.

 

The following users can use the Setup ULD Stock screen.

·        The ULD controller in the airline network

·        Terminal Operations Staff

Setup maximum and minimum limits of ULD

 

1.      Select the ULD Management tab. On the Stock Management menu, click Setup Uld Stock to get the Setup ULD Stock screen.

 

2.      In the Setup ULD Stock screen, type or select the Station or Airline Code. (mandatory)

Note:

·        If you login with a GHA privilege, the login airport appears in the Station field . This cannot be edited.

·        If you login with airline privilege, the own airline code appears in the Airline Code field. This cannot be edited.

 

3.      Click List. (mandatory)

The list of ULD stocks if already exists, is displayed in the table section.

 

4.      Click the Create link. (mandatory)

The Add Stock screen appears.

 

5.      In the Add Stock screen:

a)    Specify the details of the ULD stock. The following are mandatory:

·        Type or select the Airline Code.

·        Type or select the ULD Type Code.

·        Select the ULD Nature .

·        Type the Min Qty.

·        Type the Std Qty.

Note:

·        The Airline Code and the Station cannot be edited.

·        To discontinue the process and return to the Setup ULD Stock screen, click Close.

b)    Click Save (mandatory)

The Setup ULD Stock screen reappears with the details of the ULD in the table

Field/Button Description – Add Stock screen

 

6.      To exit the screen, click Close.

Field/Button Description – Setup ULD Stock screen

View list of ULD stock

 

1.      Select the ULD Management tab. On the Stock Management menu, click Setup Uld Stock to get the Setup ULD Stock screen.

 

2.      In the Setup ULD Stock screen, type or select the Station or Airline Code. (mandatory)

Note:

·        If you login with a GHA privilege, the login airport is displayed in the Station field. This cannot be edited.

·        If you login with an airline privilege, the own airline code is displayed in the Airline Code field. This cannot be edited.

 

3.      Click List. (mandatory)

The list of ULD stocks that satisfy the search criteria appear in the table.

 

4.      To exit the screen, click Close.

Field/Button Description – Setup ULD Stock screen

 

Modify details of ULD stock

 

1.      Select the ULD Management tab. On the Stock Management menu, click Setup Uld Stock to get the Setup ULD Stock screen.

 

2.      In the Setup ULD Stock screen, list the ULD stocks. (mandatory)

 

3.      Select the row to be modified. (mandatory)

Note:

You can modify multiple ULDs at a time. To do this, select all the corresponding rows.

 

4.      Click the Modify link.( mandatory)

The Add Stock screen appears.

 

5.      In the Add Stock screen,

a)    Make the required changes. The following cannot be edited.

·        Airline Code, Station

·        ULD Nature

·        ULD Group Code

Note:

To discontinue the process and return to the Setup ULD Stock screen, click Close.

b)    Click Save . (mandatory)

The modifications are saved and the Setup ULD Stock screen reappears.

Field/Button Description – Add Stock screen

 

6.      To exit the screen, click Close.

Field/Button Description – Setup ULD Stock screen

 

Print details of the ULD stock

 

1.      Select the ULD Management tab. On the Stock Management menu, click Setup Uld Stock to get the Setup ULD Stock screen.

 

2.      In the Setup ULD Stock screen, list the ULD stocks. (mandatory)

 

3.      Click Print. (mandatory)

The list of ULD stocks is printed.

 

4.      To exit the screen, click Close.

Field/Button Description – Setup ULD Stock screen

 

Delete details of ULD stock

 

1.      Select the ULD Management tab. On the Stock Management menu, click Setup Uld Stock to get the Setup ULD Stock screen.

 

2.      In the Setup ULD Stock screen, list the ULDs stocks. (mandatory)

 

3.      In the table, select the required row. (mandatory)

Note:

You can delete multiple ULD stocks at a time. To do this, select all the corresponding rows.

 

4.      Click Delete. (mandatory)

Click Yes in the confirmation message that appears.

The selected row gets deleted from the system and the updated table appears.

 

5.      To exit the screen, click Close.
