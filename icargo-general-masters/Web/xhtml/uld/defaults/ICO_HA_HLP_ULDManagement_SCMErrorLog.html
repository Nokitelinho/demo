To access the SCM Error Log screen:
Select the ULD Management tab. On the ULD Messages menu, click SCM Error logs.

When you access the screen, default values are displayed in the following fields:

·        Airport. – logged in station

·        Stock Chk Date(Stock Check Date). - current date

 

The screen provides links to the SCM ULD Error Log screen.

 

The following users can use the SCM Error Log screen.

·        ULD Controller

·        Ground Handling Staff

View list of SCMs generated

 

1.      Select the ULD Management tab. On the ULD Messages menu, click SCM Error logs to get the SCM Error Log screen.

 

2.      In the SCM Error Log screen, specify the search criteria. The following are mandatory:

·        Type or select the Airport.

·        Type or select the Airline .

Note :

When you login as an airline user, it will be the airport code and stock check date that will be defaulted but if you are logging in as a GHA user, it will be the airline code that will be defaulted.

 

3.      Click List. (mandatory)

The list of SCMs generated for the specified airline from the logged in station appears in the table.

 

4.      To exit the screen, click Close.

View details of an SCM

 

1.      Select the ULD Management tab. On the ULD Messages menu, click SCM Error logs to get the SCM Error Log screen.

 

2.      In the SCM Error Log screen, specify the search criteria. The following are mandatory:

·        Type or select the Airport.

·        Type or select the Airline .

Note :

When you login as an airline user, it will be the airport code and stock check date that will be defaulted but if you are logging in as a GHA user, it will be the airline code that will be defaulted.

 

3.      Click List. (mandatory)

The list of SCMs generated for the specified airline from the logged in station appears in the table.

 

4.      Select the SCM whose details you wish to view.

 

5.      Click Details. (mandatory)

The SCM ULD Error Log screen appears.

 

6.      To exit the screen, click Close.

View SCM ULD errors

 

1.      Select the ULD Management tab. On the ULD Messages menu, click SCM Error logs to get the SCM Error Log screen.

 

2.      In the SCM Error Log screen, specify the search criteria. The following are mandatory:

·        Type or select the Airport.

·        Type or select the Airline .

 

Note :

When you login as an airline user, it will be the airport code and stock check date that will be defaulted but if you are logging in as a GHA user, it will be the airline code that will be defaulted.

 

3.      Click List. (mandatory)

The SCMs that match the specified search criteria appears in the table.

 

4.      Select the SCM whose errors are to be viewed.

 

5.      In the Reconcile field of the selected SCM, click View ULD Error. (mandatory)

The SCM ULD Error Log screen will appear with the ULDs against which there are errors.

 

6.      To exit the screen, click Close.

Send an SCM

 

1.      Select the ULD Management tab. On the ULD Messages menu, click SCM Error logs to get the SCM Error Log screen.

 

2.      In the SCM Error Log screen, specify the search criteria. The following are mandatory:

·        Type or select the Airport.

·        Type or select the Airline .

 

Note :

When you login as an airline user, it will be the airport code and stock check date that will be defaulted but if you are logging in as a GHA user, it will be the airline code that will be defaulted.

 

3.      Click List. (mandatory)

The list of SCMs generated for the specified airline from the logged in station appears.

 

4.      Click Send. (mandatory)

The message is sent to the configured addresses.

 

5.      To exit the screen, click Close.
