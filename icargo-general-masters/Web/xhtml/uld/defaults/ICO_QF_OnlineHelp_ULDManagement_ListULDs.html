List ulds

(Screen ID – ULD006)

Click here to view screen access information.

To access the List ULDs screen:
Select the ULD Management tab. On the Stock Management menu, click List ULD.

RELATED TOPICS

Create a ULD

 

When you access the screen,

·        If you are an airline user, the Operating Airline code is displayed by default in the Airline Code field.

·        If you are a GHA (Ground Handling Agent) user, the login station is displayed by default in the Current Airport field.

 

The screen provides links to Maintain ULD, Record ULD Movement, Loan/Borrow Details, Maintain Damage Report, Maintain Discrepancy and Generate SCM screens.

 

The following users can use the List ULDs screen.

·        Ground Handling Agent

·        Airline User

Note:

·        To create a new ULD: Click Create in the List ULDs screen. This takes you to the Maintain ULD screen, where you can create a new ULD.

 

·        To view/modify ULD details:

-   If you are not aware of the ULD Number of the ULD, you can use the List ULDs screen. In this screen, provide your search criteria, locate the required ULD and then view/modify its details.

 

-   However, if you know the ULD Number of the ULD, you can use the Maintain ULD screen. Specify the ULD Number of the ULD, and then display its details for viewing/modification.

3.      Click List. (mandatory)

Modify details of a ULD

 

1.      Select the ULD Management tab. On the Stock Management menu, click List ULD to get the List ULDs screen.

 

2.      In the List ULDs screen, list the ULDs. (mandatory)

 

3.      In the ULD Details table, select the required row. (mandatory)

 

4.      Click Details. (mandatory)

The Maintain ULD screen appears with the details of the ULD.

 

5.      In the Maintain ULD screen:

a)    Make the required changes.

Modify details of a ULD

b)    Click Save.

The modifications are saved.

c)    Click Close.

The List ULDs screen reappears with the updated ULD details.

Field/ Button Description – Maintain ULD screen

 

6.      To exit the screen, click Close.

Delete a ULD from the system

 

1.      Select the ULD Management tab. On the Stock Management menu, click List ULD to get the List ULDs screen.

 

2.      In the List ULDs screen, list the ULDs. (mandatory)

 

3.      In the ULD Details table, select the ULD to be deleted. (mandatory)

Note:

You can delete multiple ULDs at a time. To do this, select the corresponding rows.

 

4.      Click Delete. (mandatory)

Click Yes in the confirmation message that appears. The selected ULD is deleted from the system.

 

5.      To exit the screen, click Close.

Field/Button Description – List ULDs screen

Relocate a ULD

 

1.      Select the ULD Management tab. On the Stock Management menu, click List ULD to get the List ULDs screen.

 

2.      In the List ULDs screen, list the ULDs. (mandatory)

 

3.      In the ULD Details table, select the required row. (mandatory)

 

4.      Click Relocate ULD. (mandatory)

The Relocate ULD screen appears.

Note:

·        To discontinue the process and return to the List ULDs screen, click Close.

·        If multiple rows are selected, the current location of all ULDs should be the same.

 

5.      In the Relocate ULD screen:

a)       Specify the required details. The following are mandatory:

·                Select the Facility Type.

·                Type or select the To Location.

Note:

The current location of the ULD is displayed in the From Location field.

 

b)       Click Save.( mandatory)

The selected ULD(s) is relocated and the List ULDs screen reappears with the new location details.

Field/Button Description – Relocate ULD screen

 

6.      To exit the screen, click Close.
