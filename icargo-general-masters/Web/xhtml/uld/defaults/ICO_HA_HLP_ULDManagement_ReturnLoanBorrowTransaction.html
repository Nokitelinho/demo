Click here to view screen access information.

The Return Loan Borrow Transaction screen is accessed from the Loan/Borrow Details Enquiry screen.
To access it:
- ULD Management > Transaction Management > Loan/Borrow Details Enquiry to get the Loan/Borrow Details Enquiry screen.
- In the Loan/Borrow Details Enquiry screen, specify the required details and click List.
- In the table, select the ULD with Txn Type as ‘Borrow’ and click Return.

 

The Return Loan Borrow Transaction screen is used to return the borrowed ULDs and accessories.

 

DESCRIPTION OF SCREEN ELEMENTS

Screen elements (fields, buttons, links and so on) are sequenced in the table as per the ‘tab order ‘on the screen. (Tab order is the sequence in which the focus changes when you press the Tab key. This is usually left to right within each row of screen elements).
