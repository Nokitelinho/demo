MaINTAIN ULD AGREEMENT

(Screen ID – ULD014)

Click here to view screen access information.

To access the Maintain ULD Agreement screen:
Select the ULD Management tab. On the Transaction Management menu, click Maintain ULD Agreement

 

When you access the screen, default values are displayed in the following fields:

·        Party Type – ‘Airline’

·        Transaction Type – ‘Borrow’

·        Agreement Status – ‘ACTIVE’

·        Agreement Date – the current date

·        Free Loan Period – ‘0’

·        Demurrage Rate – ‘0.0’

·        Taxes – ‘0.0’

·        Demurrage Frequency – ‘Daily’

 

The ULD Controller can use the Maintain ULD Agreement screen.

Functions

Create a ULD agreement

View details of an existing ULD agreement

Modify details of an existing ULD agreement

Print the ULD agreement

 

Note:

To view/modify a ULD Agreement:

·        If you are not aware of the Agreement Number, you can use the List Agreement screen. In this screen, provide your search criteria, locate the required agreement and then view/modify details.

·        However if you know the Agreement Number, you can use the Maintain ULD Agreement screen. Specify the Agreement Number and then display its details for viewing/modification.

Create a ULD agreement

 

1.      Select the ULD Management tab. On the Transaction Management menu, click Maintain ULD Agreement to get the Maintain ULD Agreement screen.

 

2.      In the Maintain ULD Agreement screen, type or select the Agreement Number. (mandatory)

 

3.      Click List. (mandatory)

·             If the agreement already exists (with the specified Agreement Number), its details appear on the screen.

·             If the agreement does not exist, a message appears enquiring whether you wish to proceed with the creation of the agreement . Click Yes in the confirmation message, and proceed with the steps given below.

 

4.      In the Maintain ULD Agreement screen, specify the Agreement Details, which is a combination of the Party Details and the Demurrage Details.

 

5.      In the Party Details section, specify the party details. The following are mandatory:

·             Type or select the Party Code.

·             Type or select the Agreement Date .

Note:

The Party Code would correspond to the Party Type selected. That is, if the Party Type is ‘Airline’, then the Party Code values populated would be airlines.

 

6.      In the Demurrage Details section, type or select the From Date. (mandatory)

 

7.      Click the Create link.

The Add Agreement screen appears.

 

8.      In the Add Agreement screen:

a)    Specify the details of the exception to be included in the agreement. The following are mandatory:

-   ULD Type

-   Airport

-   Valid From

Note:

·        To add more agreement details, click the Create link.

·        Click the First|Prev|Next|Last links to navigate to the other agreement details.

·        To delete the agreement details, click the Delete link.

b)    Click Ok.

The Maintain ULD Agreement screen reappears. The agreement appears in the table section.

Field/Button Description – Add Agreement screen

 

9.      Click Save. (mandatory)

The agreement details are saved and an Agreement Number is generated.

 

10.   To exit the screen, click Close.

Modify details of an existing ULD agreement

If you know the Agreement Number, you can use the Maintain ULD Agreement screen. Specify the Agreement Number and then display its details for viewing/modification.

 

Note:

If you are not aware of the Agreement Number, you can use the List Agreement screen. In this screen, provide your search criteria, locate the required agreement and then view/modify details.

 

View list of ULD agreement details

Locate a ULD agreement and view/modify details

 

1.      Select the ULD Management tab. On the Transaction Management menu, click Maintain ULD Agreement to get the Maintain ULD Agreement screen.

 

2.      In the Maintain ULD Agreement screen, type or select the Agreement Number. (mandatory)

 

3.      Click List. (mandatory)

The details of the agreement appear on the screen.

 

4.      Make the required changes in the Maintain ULD Agreement screen.

a)    The following fields are mandatory in the Party Details section:

·        Type or select the Party Code.

·        Type or select the Agreement Date .

 

Note:

The Party Code would correspond to the Party Type selected. That is, if the Party Type is ‘Airline’, then the Party Code values populated would be ‘Airlines’.

 

b)    In the Maintain ULD Agreement screen, in the Demurrage Details section, type or select the From Date. (mandatory)

Note:

You can also specify the day from which the contract would come in effect, the demurrage rates/ taxes that would apply and the frequency at which it would be calculated.

 

5.      Select the exception which you wish to modify and click the Modify link.

The Add Agreement screen appears.

 

6.      In the Add Agreement screen:

a)    Make the required changes.

b)    The following fields are mandatory:

-   ULD Type

-   Airport

-   Valid From

c)    Click Ok. (mandatory)

The Maintain ULD Agreement screen reappears.

Field/Button Description – Add Agreement screen

 

Note:

You can delete an exception. For this:

a)    Click the Delete link. Click Yes in the confirmation message that appears. (mandatory)

The selected exception is deleted.

b)    Click Save. (mandatory)

The modifications made are saved to the ULD agreement.

 

7.      Click Save. (mandatory)

The details of the specified Agreement Number are saved.

 

8.      To exit the screen, click Close.
