To access the Maintain Accessories screen:
Select the ULD Management tab. On the Stock Management menu, click Maintain Accessory.

RELATED TOPICS

View list of accessory stocks

 

When you access the screen, default values are displayed in the following fields:

·          Acc.Code(Accessory Code) – ‘BELT’

·          Airline Code – the default airline code

 

The ULD Controller can use the Maintain Accessories screen.

Note:

To view/modify accessory stock details:

-        If you are not aware of the Accessory Code, Airline Code and Airport of the accessory stock, you can use the List Accessory Stock screen. In this screen, provide your search criteria, locate the required accessory stock and then view/modify its details.

 

Locate an accessory stock and view/modify its details

 

-        However, if you know the Accessory Code, Airline Code and Airport of the accessory stock, you can use the Maintain Accessories screen. Specify the above mandatory details and then display its details for viewing/modification.

 

Create an accessory

 

1.      Select the ULD Management tab. On the Stock Management menu, and click Maintain Accessory to get the Maintain Accessories screen.

 

2.      In the Maintain Accessories screen, specify the search criteria. The following are mandatory:

·        Select the Acc.Code (Accessory Code)

·        Type or select the Airport .

Note:

·          If you login as an airline user, the default airline code is displayed in the Airline Code field and cannot be edited.

·          If you login as GHA user, the logged in airport is displayed in the Airport field and cannot be edited.

 

3.      Click List. (mandatory)

Click Yes in the confirmation message that appears.

 

4.      Type the Min.Qty. (Minimum Quantity). (mandatory)

 

5.      Click Save. (mandatory)

The details of the accessory are saved.

 

6.      To exit the screen, click Close.

Modify details of an accessory

If you are not aware of the Accessory Code, Airline Code and Airport of the accessory stock, you can use the List Accessory Stock screen. In this screen, provide your search criteria, locate the required accessory stock and then view/modify its details.

Locate an accessory stock and view/modify its details

 

1.      Select the ULD Management tab. On the Stock Management menu, and click Maintain Accessory to get the Maintain Accessories screen.

 

2.      In the Maintain Accessories screen, specify the search criteria. The following are mandatory:

·        Select the Acc.Code (Accessory Code)

·        Type or select the Airport .

 

3.      Click List. (mandatory)

The details of the accessory appear on the screen.

 

4.      Make the required changes. The following fields cannot be edited

·        Acc.Code

·        Airline Code

·        Airport

All the other fields in the screen can be edited.

 

5.      Click Save. (mandatory)

The modifications are saved in the system.

 

6.      To exit the screen, click Close.
