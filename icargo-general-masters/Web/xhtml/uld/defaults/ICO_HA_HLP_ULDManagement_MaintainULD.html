Click here to view screen access information.

To access the Maintain ULD screen:
Select the ULD Management tab. On the Stock Management menu, click Maintain ULD.

 

When you access the screen, default values are displayed in the following fields:

·        Overall Status – ‘Operational’

·        Cleanliness Status – ‘Clean’

·        Damage Status – ‘Not Damaged’

·        ULD Nature – ‘General’

 

The screen provides a link to the ULD History screen.

 

The ULD Controller can use the Maintain ULD screen.

Note:

To view/modify details of a ULD:

·        If you are not aware of the ULD No., you can use the List ULD screen. In this screen, provide your search criteria, locate the required ULD and then view/modify its details.

 

·        However, if you know the ULD No., you can use the Maintain ULD screen. Specify the ULD No., and then view/modify its details.

Create a ULD

 

1.      Select the ULD Management tab. On the Stock Management menu, click Maintain ULD to get the Maintain ULD screen.

 

2.      In the Maintain ULD screen, type the ULD Numberof the new ULD. (mandatory)

 

3.      Click List. (mandatory)

·        If the ULD already exists (with the specified ULD number), its details appear on the screen.

·        If the ULD does not exist, a message appears enquiring whether you wish to create a new one. Click Yes in this confirmation message, and proceed with the following steps.

 

4.      In the ULD Details section, specify the Structural Details of the ULD. The following are mandatory:

·        Type the Tare Wt.

·        Type the Structural Wt. Limit .

·        Type the Base Dim. L (Base Dimensional Length).

·        Type the Base Dim. H (Base Dimensional Height).

·        Type the Base Dim. W (Base Dimensional Width).

 

Note:

·        The details in this section are populated based on the structural details of the ULD Type as maintained within the ULD Type Masters.

·        Any ULD type that is not maintained in the system, if specified will display an error message stating that it is invalid.

 

5.      In the ULD Details section, specify the Operational Details of the ULD.

a)    Type or select the Current Airport. (mandatory)

b)    Specify the remaining operational details, as required. (optional)

Note:

The ULD owner airline code appears in the Airline Codefield.

 

6.      In the ULD Details section, specify the Asset Details of the ULD, as required. (optional)

 

Note:

·        In the Maintain ULDscreen, the Movement History and Delete buttons become enabled only for ULDs that are already maintained within the system .

·        The IATA Replacement Cost is populated based on the value provided in the ULD Type Setup screen.

·        The Current Value field is non-editable and it changes as per the depreciation rate maintained internally.

Create ULD type

 

7.      Type the Remarks , if any. (optional)

 

8.      Click Save. (mandatory)

The ULD details are saved.

 

9.      To exit the screen, click Close.

Field/Button Description – Maintain ULD screen

 

Create multiple ULDs simultaneously

 

1.      Select the ULD Management tab. On the Stock Management menu, click Maintain ULD to get the Maintain ULD screen.

 

2.      Select the Create Multiple check box to create multiple ULDs. (mandatory)

 

3.      Specify the details of the multiple ULDs to be created. The following are mandatory:

·        Type the ULD Type Code

·        Type the Airline Code

 

4.      You can specify the range of the multiple ULDs required. (mandatory)

To do this, click the Specify Range button. The Create Multiple ULDs screen appears.

 

5.      In the Create Multiple ULDs screen,

 

a)    In the ULD Details section, specify the following mandatory details:

·        Type the Start No. (mandatory)

·        Type the No. of units. (mandatory)

b)    Click the Generatebutton.  ULD numbers are auto generated for the specified ULD type and total number of units, starting with the Start No . provided.

Note:

From the generated list of ULD numbers, you can do the following:

Add ULD numbers:

To do this:

i)     Click the Add link at the top right corner of the table. A new row appears in the table.

ii)    In the row, type the ULD No.

A ULD number is added.

Delete ULD numbers:

To do this:

i)     Select a row.

ii)    Click the Delete link on the top right corner of the table.

The selected row is deleted.

 

c)    Click OK.

The Maintain ULD screen reappears.

d)    To exit the Create Multiple ULDs screen, click Close.

 

6.      In the ULD Details section, specify the Structural Details of the ULD. The following are mandatory:

·        Type the Tare Wt.

·        Type the Structural Wt. Limit .

·        Type the Base Dim. L (Base Dimensional Length).

·        Type the Base Dim. H (Base Dimensional Height).

·        Type the Base Dim. W (Base Dimensional Width).

 

Note:

·        The details in this section are populated based on the structural details of the ULD Type as maintained within the ULD Type Masters.

·        Any ULD type that is not maintained in the system, if specified will display an error message stating that it is invalid.

 

7.      In the ULD Details section, specify the Operational Details of the ULD.

e)    Type or select the Current Airport. (mandatory)

f)     Specify the remaining operational details, as required. (optional)

Note:

The ULD owner airline code appears in the Airline Codefield.

 

8.      In the ULD Details section, specify the Asset Details of the ULD, as required. (optional)

 

Note:

·        In the Maintain ULDscreen, the Movement History and Delete buttons become enabled only for ULDs that are already maintained within the system .

·        The IATA Replacement Cost is populated based on the value provided in the ULD Type Setup screen.

·        The Current Value field is non-editable and it changes as per the depreciation rate maintained internally.

Create ULD type

 

9.      Click Save.

The multiple ULDs are saved.

 

10.   To exit the screen, click Close.

Field/Button Description – Maintain ULD screen

 

Modify details of a ULD

If you know the ULD No., you can use the Maintain ULD screen. Specify the ULD No. and then display ULD details for modification.

 

Note:

However, if you are not aware of the ULD No. of the booking, go to the List ULD screen to search for, locate and then modify details of the ULD.

View list of ULDs

Locate a ULD and view/modify ULD details

 

 

1.      Select the ULD Management tab. On the Stock Management menu, and click Maintain ULD. to get the Maintain ULD screen

 

2.      In the Maintain ULD screen, type the ULD Number . (mandatory)

 

3.      Click List. (mandatory)

The details of the specified ULD Number appear in the ULD Details section.

 

4.      Make the required changes.

a)    In the ULD Details section, modify the Structural Details of the ULD, as required. The following are mandatory fields:

·        Type the Tare Wt.

·        Type the Structural Wt. Limit .

·        Type the Base Dim. L (Base Dimensional Length).

·        Type the Base Dim. H (Base Dimensional Height).

·        Type the Base Dim. W (Base Dimensional Width).

In the ULD Details section, modify the Operational Details of the ULD, as required. The Current Airport is a mandatory field.

Note:

The Airline Code cannot be edited.

b)    In the ULD Details section, modify the Asset Details of the ULD, as required.

c)    Type/modify the Remarks,if any.

 

5.      Click Save.

The modifications are saved.

 

6.      To exit the screen, click Close.

Field/Button Description – Maintain ULD screen

 

Delete a ULD

 

1.      Select the ULD Management tab. On the Stock Management menu, click Maintain ULD to get the Maintain ULD screen.

 

2.      In the Maintain ULD screen, type the ULD Number . (mandatory)

 

3.      Click List. (mandatory)

The details of the specified ULD Number appear in the ULD Details section.

 

4.      Click Delete. Click Yes in the confirmation message that appears. (mandatory)

The ULD is deleted from the system.

 

5.      To exit the screen, click Close.

Field/Button Description – Maintain ULD screen

 

View movement history of a ULD

You can view the movement details (both internal and external), damage/repair history, transaction details, and build-up/break-down details of ULDs.

 

1.      Select the ULD Management tab. On the Stock Management menu, and click Maintain ULD to get the Maintain ULD screen.

 

2.      In the Maintain ULD screen, type the ULD Number . (mandatory)

 

3.      Click List. (mandatory)

The details of the specified ULD Number appear in the ULD Details section.

 

4.      Click Movement History. (mandatory)

The ULD Movement History screen appears with the movement details of the specified ULD.

List history details of a ULD

 

5.      To exit the screen, click Close.

The Maintain ULD screen reappears.
