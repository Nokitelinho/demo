To access the Send UCM IN/OUT Message screen:
Select the ULD Management tab. On the ULD Messages menu, click Send UCM IN/OUT Message.

 

When you access the screen, default values are displayed in the following fields:

·        UCM OUT – This radio button appears selected.

·        Origin - the logged in station

 

The ULD Controller can use the Send UCM IN/OUT Message screen.

Functions

Generate UCM OUT message

Generate UCM IN message

Generate UCM OUT message

 

1.      Select the ULD Management tab. On the ULD Messages menu, click Send UCM IN/OUT Messageto get the Send UCM IN/OUT Message screen.

 

2.      In the Send UCM IN/OUT Message screen, in the Flight Details section, specify the search criteria. The following are mandatory:

·        Type the Flight No.

·        Type or select the Flight Date.

 

3.      Click List. (mandatory)

Note:

·        The View CPM ULDs button is enabled.

·        The flight route and the status of the flight are displayed in the section below the Flight Details section.

·        The possible destinations for the flight appear in the Destination drop-down list.

 

4.      Select a particular Destination and click View CPM ULDs. (mandatory)

If the CPM (Container Palette message) corresponding to the message is already processed, then the details of the ULDs will be listed. The Add| Delete links are enabled.

 

5.      Click the Add link. (mandatory)

A new row appears in the ULDS table.

 

6.      In the ULDS table, specify the details of the ULD.

Type the ULD NO . This is mandatory

Note:

·        You can add more ULDs by clicking the Add link.

·        You can delete ULD from a flight. To do this select the ULD and click the Delete link.

 

7.      Click Send. (mandatory)

The UCM OUT message is sent.

 

8.      To exit the screen, click Close.

Field/Button Description – Send UCM IN/OUT Message screen

 

Generate UCM IN message

 

1.      Select the ULD Management tab. On the ULD Messages menu, click Send UCM IN/OUT Messageto get the Send UCM IN/OUT Message screen.

 

2.      In the Send UCM IN/OUT Message screen, select the UCM IN. (mandatory)

 

3.      In the Send UCM IN/OUT Message screen, in the Flight Details section, specify the search criteria. The following are mandatory:

·        Type the Flight No.

·        Type or select the Flight Date.

 

4.      Click List. (mandatory)

Note:

·        The View CPM ULDs button is enabled.

·        The flight route and the status of the flight are displayed in the section below the Flight Details section.

·        The possible destinations for the flight appear in the Destination drop-down list.

 

5.      Click View CPM ULDs. (mandatory)

The Add| Delete links are enabled.

 

6.      Click the Add link. (mandatory)

A new row appears in the ULDS table.

 

7.      In the ULDS table, type the ULD NO. (mandatory)

 

8.      Click Send. (mandatory)

The UCM IN message is sent.

 

9.      To exit the screen, click Close.
