Record discrepancy or create discrepancy for a newly found ULD

 

1.      Select the ULD Management tab. On the ULD Tracking menu, click Maintain Discrepancy to get the Maintain Discrepancy screen.

 

2.      In the Maintain Discrepancy screen, specify the criteria . The following are mandatory:

·        Type the ULD No.

·        Type or select the Discrepancy Date.

·        Type or select the Reporting Airport.

·        Type or select the Location.

 

3.      Select the Discrepancy Code as ‘Found’.

 

4.      Click Save.

The details are saved.

 

5.      To exit the screen, click Close.
