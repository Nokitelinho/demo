POOL OWNERSHIP MASTER

(Screen ID –ULD025)

Click here to view screen access information.

To access the Pool Ownership Master screen:
Select the ULD Management tab. On the Stock Management menu, click ULD Pool Owners .

 

When you access the screen, the default airline code is displayed in the Airline 1 field in the Participating Airlines section.

 

The ULD Controller in the airline network can use the Pool Ownership Master screen.

 

The following users can use the Maintain Flight Schedule screen.

·        Route Controller of the flight controlling station

·        Head Quarters

Create pool ownership

 

1.      Select the ULD Management tab . On the Stock Management menu, click ULD Pool Owners to get the Pool Ownership Master screen.

 

2.      In the Pool Ownership Master screen, click the Add link.   ( mandatory )

A new row appears in the Pool Owner Details section.

 

3.      In the new row:

a)    Specify the following mandatory details.

·        Type or select Airline1.

·        Type or select Airline2.

·        Type or select Airport.

Note:

If no airport is specified, then ‘NA’ (not applicable) appears in Airport.

b)    You can type Remarks, if required. ( optional)

You can add more pool ownerships. To do this, click the Add link and repeat the above steps. ( optional )

 

4.      Click Save. ( mandatory )

The specified pool ownerships are saved.

 

5.      To exit the screen, click Close.

Note:

Once a pool ownership is created, it cannot be modified. However you can modify the Remarks recorded against it. You can add more pool ownerships to the list or delete existing ones.

Field/Button Description-Pool Ownership Master screen

 

View list of pool ownerships

 

1.      Select the ULD Management tab . On the Stock Management menu, click ULD Pool Owners to get the Pool Ownership Master screen.

 

2.      In the Pool Ownership Master screen, specify the search criteria, as required. None are mandatory.

 

3.      Click List. ( mandatory)

The airlines participating in the pool ownership appear in the Pool Owner Details section.

 

Note:

To view all the pool owner details, click List without specifying any search criteria.

 

4.      To exit the screen, click Close.

Field/Button Description-Pool Ownership Master screen

 

Delete a pool ownership

 

1.      Select the ULD Management tab . On the Stock Management menu, click ULD Pool Owners to get the Pool Ownership Master screen.

 

2.      In the Pool Ownership Master screen, list the pool ownerships. ( mandatory)

 

In the Pool Ownership Details section, select the row to be deleted. ( mandatory )

 

3.      Click the Delete link. ( mandatory )

 

4.      Click Save. ( mandatory )

The pool ownership is deleted from the system.

 

5.      To exit the screen, click Close.

Field/Button Description-Pool Ownership Master screen

 

Define segment exception for a pool ownership

 

If there are any segments for which demurrages are to be charged even if there a pool owner agreement in place, then these can be specified.

 

1.      Select the ULD Management tab . On the Stock Management menu, click ULD Pool Owners to get the Pool Ownership Master screen.

 

2.      In the Pool Ownership Master screen, list the pool ownerships. ( mandatory)

 

3.      In the Pool Ownership Details section, select the required row. ( mandatory )

 

4.      Click SegmentExceptions. ( mandatory )

The SegmentExceptions screen appears.

 

In the SegmentExceptions screen:

a)    Click the Add link. ( mandatory)

A new row appears.

b)    In the row, specify the following mandatory details.

·        Type or select the Origin.

·        Type or select the Destination.

 

c)    You can define more segment exceptions. To do this, click the Add link and repeat the above step. ( optional)

Note:

To delete a segment exception, select the row and click the Delete link.

d)    Click OK. ( mandatory )

The Pool Ownership Master screen reappears.

Field/Button Description-Segment Exception screen

 

5.      Click Save.

The segment exception is saved.

 

6.      To exit the screen click Close.
