Record dummy movement of a ULD

1.      Select the ULD Management tab. On the ULD Tracking menu, click Record ULD Movement to get the Record ULD Movement screen.

 

2.      In the Record ULD Movement screen: In the Flight Details table select the Dummy Movement checkbox. (mandatory)

 

3.      In the Record ULD Movement screen: In the Flight Details table, specify the required flight details. The following are mandatory.

·        Type or select the POL.

·        Type or select the POU.

Note:

To delete the flight details, select the required row and click the Delete link.

 

4.      Click the Add link in the ULD Details table. (mandatory)

A new row appears in the ULD Details table.

 

5.      In the new row that is added, type the ULD No. (mandatory)

 

6.      You can add more ULDs. To do this, repeat the above steps.

Note:

Select the Update Current Airport checkbox, if you want to update the stock of the ULD to the current airport. The current airport appears in the Current Airport field .

 

7.      Click Save. (mandatory)

The specified details are saved.

 

8.      To exit the screen, click Close.

Record actual movement of a ULD

 

1.      Select the ULD Management tab. On the ULD Tracking menu, click Record ULD Movement to get the Record ULD Movement screen.

 

2.      In the Record ULD Movement screen: In the Flight Details table specify the required flight details. The following are mandatory.

·        Type or select the Carrier Code.

·        Type the No.

·        Type or select the Flight Date.

·        Type or select the POL.

·        Type or select the POU.

 

3.      Click the Add link in the ULD Details table. (mandatory)

A new row appears in the ULD Details table.

 

4.      In the new row that is added, type the ULD No. (mandatory)

 

5.      You can add more ULDs. To do this, repeat the above steps.

Note:

Select the Update Current Airport checkbox, if you want to update stock of the ULD to the current airport. The current airport appears the Current Airport field .

 

6.      Click Save. (mandatory)

The specified details are saved.

 

7.      To exit the screen, click Clos e.
