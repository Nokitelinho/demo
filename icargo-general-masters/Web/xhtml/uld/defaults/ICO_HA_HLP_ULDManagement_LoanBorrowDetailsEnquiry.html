ULD MANAGEMENT

Loan/Borrow Details Enquiry

(Screen: ULD009)

Click here to view screen access information.

To access the Loan/Borrow Details Enquiry screen:
Select the ULD Management tab. On the Transaction Management menu, click Loan/Borrow Details Enquiry.

 

The screen provides link to Maintain ULD and Loan/Borrow Details screen.

 

The Loan/Borrow Details Enquiry screen can be used by the ULD Controller.

View list of loan/borrow details

 

1.      Select the ULD Management tab. On the Transaction Management menu, click Loan/Borrow Details Enquiry to get the Loan/Borrow Details Enquiry screen.

 

2.      In the Search Criteria section, specify the required search criteria. None are mandatory.

 

3.      Click List. (mandatory)

The list of loan/borrow details that match the search criteria appear in the table below.

 

4.      To exit the screen, click Close

Field/Button Description – Loan/Borrow Details Enquiry screen

 

View / Modify ULD transactions details

 

1.      Select the ULD Management tab. On the Transaction Management menu, click Loan/Borrow Details Enquiry to get the Loan/Borrow Details Enquiry screen.

 

2.      In the Loan/Borrow Details Enquiry screen, list the details. (mandatory)

 

3.      In the ULD Transaction Details tab , select a row. (mandatory)

 

4.      Click Modify Txn (mandatory)

The Modify ULD Transaction Details screen appears.

 

5.      In the Modify ULD Transactions Details screen, make the required changes.

a)    The following fields cannot be edited:

·        ULD No.

·        Txn Type

·        PartyType

·        Party Code

·        CRN.

Note:

To modify the transaction, the transaction status should be ‘To be returned’ or ‘To be invoiced’

b)    Click Ok.

The details are saved and the Loan Borrow Details Enquiry screen reappears with the updated details

 

6.      To exit the screen, click Close.

Field/Button Description – Loan/Borrow Details Enquiry screen

 

Print UCR

 

1.      Select the ULD Management tab. On the Transaction Management menu, click Loan/Borrow Details Enquiry to get the Loan/Borrow Details Enquiry screen.

 

2.      In the Loan/Borrow Details Enquiry screen, list the details. (mandatory)

 

3.      Click Print UCR. (mandatory)

The ULD Control Receipt is printed.

 

4.      To exit the screen, click Close

Field/Button Description – Loan/Borrow Details Enquiry screen

 

Return loaned/borrowed ULDs

 

1.      Select the ULD Management tab. On the Transaction Management menu, click Loan/Borrow Details Enquiry to get the Loan/Borrow Details Enquiry screen.

 

2.      In the Loan/Borrow Details Enquiry screen, list the details. (mandatory)

 

3.      In the ULD Transaction Details tab, select a row (mandatory)

 

4.      Click Return  (mandatory)

The Return Loanborrow  Transaction screen appears

 

5.      In the Return Loanborrow  Transaction screen, make the required changes.

c)    The following fields cannot be edited:

·        Txn Type

·        Return Type

·        Return Airport

·        ULD No

·        Loan Period

·        Demurrage

·        Taxes

·        Total

·        Currency

·        Damaged

·        Return to

·        Remarks

 

d)    Click Save.

The ULD is returned and the Loan Borrow Details Enquiry screen reappears with the updated details

 

 

6.      To exit the screen, click Close

Field/Button Description – Loan/Borrow Details Enquiry screen

Generate Invoice

 

1.      Select the ULD Management tab. On the Transaction Management menu, click Loan/Borrow Details Enquiry to get the Loan/Borrow Details Enquiry screen.

 

2.      In the Loan/Borrow Details Enquiry screen, list the details. (mandatory)

 

 

3.      Click Generate Invoice. (mandatory)

The Generate Invoice screen appears.

a)    In the Generate Invoice screen, specify the details. The following are mandatory

·        Type or select the Invoice Date

·        Type the or select Invoice To Code

·        Type the Name

·        Type the Waiver

b)    Click Show Total. (mandatory)

The Demurrage is calculated

c)    Click Generate. (mandatory)

The invoice is generated and the Loan Borrow Details Enquiry Details screen appears with the updated details.

The status of the transaction changes to “Invoiced”.

 

4.      To exit the screen, click Close

Field/Button Description – Loan/Borrow Details Enquiry screen

 

Generate LUC

 

1.      Select the ULD Management tab. On the Transaction Management menu, click Loan/Borrow Details Enquiry to get the Loan/Borrow Details Enquiry screen.

 

2.      In the Loan/Borrow Details Enquiry screen, list the details. (mandatory)

 

3.      Click Generate LUC. (mandatory)

The New Message screen appears.

 

4.      In the New Message screen,

a)    Select the mode . (mandatory)

b)    Click Send. (mandatory)

The LUC message is send

Note:

LUC messages can be generated for ULDs with transaction status ‘To be Returned.
