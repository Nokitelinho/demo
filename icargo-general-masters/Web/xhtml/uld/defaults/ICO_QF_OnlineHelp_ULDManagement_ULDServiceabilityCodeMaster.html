ULD SERVICEABILITY CODE MASTER

(Screen ID – ULD047)

Click here to view screen access information.

To access the ULD Serviceability Code Master screen:
Select the ULD Management tab. On the Stock Management menu, click ULD Serviceability Code Master.

 

When you access the screen, ‘Agent’ appears in the Party Type field.

 

The ULD Controller in the airline network can use the ULD Serviceability Code Master screen.

Create ULD serviceability code

 

1.      Select the ULD Management tab. On the Stock Management menu, click ULD Serviceability Code Master to get the ULD Serviceability Code Master screen.

 

2.      In the ULD Serviceability Code Master screen: In the Search Criteria section , select the Party Type. This is mandatory. (mandatory)

 

3.      Click List . (mandatory)

The list of ULD serviceability codes satisfying the specified search criteria appears in the screen.

 

4.      Click the Add link. (mandatory)

A new row appears in the screen.

 

5.      Specify the serviceability details in the table. The following are mandatory,

·          Type the Code.

·          Type the Description.

 

6.      Click Save. (mandatory)

The ULD serviceability code details are saved.

 

7.      To exit the screen, click Close.
