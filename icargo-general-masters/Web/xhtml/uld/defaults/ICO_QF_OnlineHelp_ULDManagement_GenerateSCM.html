To access the Generate SCM screen:
Select the ULD Management tab. On the ULD Messages menu, click Generate SCM.

 

When you access the screen, default values are displayed in the following fields:

·        Airline - operating airline code

·        Airport – logged in station

·        Stock Check Date – current date

 

The following users can use the Generate SCM screen.

·        Ground Handling Agent

·        ULD Controller

List ULDs that are in system stock

 

1.      Select the ULD Management tab. On the ULD Messages menu, click Generate SCM to get the Generate SCM screen.

 

2.      In the Generate SCM screen, specify the search criteria. The following are mandatory:

·        Type or select the Airport .

·        Type or select the Stock Check Date .

Note:

Airline cannot be edited. The operating airline code is displayed here by default.

 

3.      Click List. (mandatory)

All the ULDs that are in the system appear in the Stock Details table.

 

4.      To exit the screen, click Close.

Field/Button Description – Generate SCM screen

 

Add a ULD to system stock

 

1.      Select the ULD Management tab. On the ULD Messages menu, click Generate SCM to get the Generate SCM screen.

 

2.      In the Generate SCM screen, list the ULDs that are in system stock. (mandatory)

 

3.      Click the Add link in the Stock Details table. (mandatory)

A new row is added in the Stock Details table.

 

4.      In the row that is added specify the following mandatory details.

·        Type the ULD No.

·        Select the Facility Type.

·        Type or select the Location.

Note:

·        The Status will be displayed as ‘new found’ and cannot be edited

·        To delete the ULD select the ULD with ‘new found’ status and click the Delete link.

 

5.      You can add more ULDs to the system stock at the same time. To do this repeat the above steps.

 

6.      Click Send. (mandatory)

The SCM is saved and an SCM sequence number is generated.

The New Message screen appears with the message.

 

7.      In the New Message screen.

a)     Specify the required details.

·        Select the Mode.

·        Type the Send To.

Note:

To discontinue the process and return to the Generate SCM screen, click Close.

b)     Click Details to add more addresses to whom SCM need to be sent.

c)      Click Send. (mandatory)

The SCM message is sent to the specified address. The Generate SCM screen reappears.

Field/Button Description – New Message screen

 

8.      To exit the screen, click Close.

Field/Button Description – Generate SCM screen

 

Mark a ULD as missing

 

1.      Select the ULD Management tab. On the ULD Messages menu, click Generate SCM to get the Generate SCM screen.

 

2.      In the Generate SCM screen, list the ULDs that are in system stock. (mandatory)

 

3.      In the Stock Details table, select the ULD you want to mark as missing. (mandatory)

 

4.       Click Mark as Missing. (mandatory)

 

5.      Click Yes in the confirmation message that appears.

The ULD number is highlighted in red and its status is changed to Missing.

 

6.      Click Send. (mandatory)

The SCM is saved with the ULD Status as Missing and a sequence number is generated.

The New Message screen appears with the message.

 

7.      In the New Message screen.

a)     Specify the required details.

·        Select the Mode.

·        Type the Send To.

Note:

To discontinue the process and return to the Generate SCM screen, click Close.

b)     Click Details to add more addresses to whom SCM need to be sent.

c)      Click Send. (mandatory)

The SCM message is sent to the specified address. The Generate SCM screen reappears.

Field/Button Description – New Message screen

Note:

To return a ULD marked as missing to the airport stock:

-   Select the required ULD and click Return to System Stock.

-   Click Yes in the confirmation message that appears.

-   The ULD number will be returned to airport stock list.

 

8.      To exit the screen, click Close.

b)     Click Details to add more addresses to whom SCM need to be sent.
