Monitor ULD Stock

(Screen ID: ULD018)

Click here to view screen access information.

To access the Monitor ULD Stock screen:
Select the ULD Management tab. On the Stock Management menu, click Monitor ULD Stock.

The Monitor ULD Stock screen is used by the ULD controller.

 

When you access the screen, default values are displayed in the following fields:

·        Airline Code - default airline code if the user has logged in using airline privileges.

·        ULD Nature -   ‘ALL’

View list of ULD stock details

 

1.      Select the ULD Management tab. On the Stock Management menu, click Monitor ULD Stock to get the Monitor ULD Stock screen.

 

2.      In the Monitor ULD Stock screen, specify the required search criteria.

Note:

·          If you are logged in as a user with airline privileges, the default carrier code appears in the Airline Code field and this is be non-editable. None of the search criteria is mandatory.

·          If you are logged in as a user with GHA privileges, it is mandatory to type or select the Airline Code.

 

3.      Click List. (mandatory)

The list of ULD stock details that satisfy the specified search criteria appear in the Stock Details table

Note:

To view the list of all ULD stock details maintained in the system, click List without specifying any search criteria.

 

4.      To exit the screen, click Close.

Field/Button Description – Monitor ULD Stock screen

View list of ULD details in a particular group or type

 

1.      Select the ULD Management tab. On the Stock Management menu, click Monitor ULD Stock to get the Monitor ULD Stock screen.

 

2.      In the Monitor ULD Stock screen, list the ULD stock details. (mandatory)

 

3.      To view list of ULDs belonging to a particular group:

In the Stock Details table, select a row.

            Or

To view list of ULDs belonging to a particular type:

a)     Click the red drop-down arrow adjacent to the Airline Code of the required ULD group.

b)     Select a row from the expanded table.

 

4.      Click List ULD. (mandatory)

·        The List ULDs screen appears with the list of ULDs of a group or a type in the stock.

View list of ULDs

·        In the List ULDs screen, click Close.

The Monitor ULD Stock screen reappears.

Field/Button Description – List ULDs screen

 

5.      To exit the screen, click Close.

Field/Button Description – Monitor ULD Stock screen

 

Setup ULD stock for an airport

1.      Select the ULD Management tab. On the Stock Management menu, click Monitor ULD Stock to get the Monitor ULD Stock screen.

 

2.      In the Monitor ULD Stock screen, specify the required search criteria. The following are mandatory.

·        Select the Level Type as ‘Airport’.

·        Type or select the Level Value.

 

3.      Click List. (mandatory)

The list of ULD stock details that satisfy the search criteria appear in the Stock Details table.

·        If stock has already been set up for the airport, its details appear in the screen.

 

4.      In the Stock Details table, select a row. (mandatory)

 

5.      Click Setup Stock . (mandatory)

·        The Setup ULD Stock screen appears with the details of the stock in the airport. You can create stock in this screen.

Setup maximum and minimum limits of ULD

·        In the Setup ULD Stock screen, click Close.

The Monitor ULD Stock screen reappears.

Field/Button Description – Setup ULD Stock screen

 

6.      To exit the screen, click Close.

Field/Button Description – Monitor ULD Stock screen

 

Generate report on the ULD stocks

 

1.      Select the ULD Management tab. On the Stock Management menu, click Monitor ULD Stock to get the Monitor ULD Stock screen.

 

2.      In the Monitor ULD screen, list the ULD stock details. (mandatory)

 

3.      Click Generate Report. (mandatory)

A report on the ULD stock is generated.

 

4.      To exit the screen, click Close.

Field/Button Description – Monitor ULD Stock screen

 

Generate SCM Message

 

1.      Select the ULD Management tab. On the Stock Management menu, click Monitor ULD Stock to get the Monitor ULD Stock screen.

 

2.      In the Monitor ULD Stock screen, list the ULD stock details. (mandatory)

 

3.      In the Stock Details table, select a row (mandatory)

 

4.      Click Generate SCM Message

An SCM (Stock Check Message) is generated for the selected ULD stock.

 

5.      To exit the screen, click Close.

Field/Button Description – Monitor ULD Stock screen

 

Print the list of ULD stock details

 

1.      Select the ULD Management tab. On the Stock Management menu, click Monitor ULD Stock to get the Monitor ULD Stock screen.

 

2.      In the Monitor ULD Stock screen, list the ULD stock details. (mandatory)

 

3.      In the Stock Details table, select a row (mandatory)

 

4.      Click Print. (mandatory)

The list of all ULD stock details is printed.

 

5.      To exit the screen, click Close.

Airline Code

field

The code of the airline whose stock is to be monitored

While you are logging in as an airline user:

·   By default, the airline code of the owner airline appears.

·   This is displayed here and cannot be edited

 

C

While you are logging in as a GHA user:

·   Type this in the field or select from the LOV.

·   To open the LOV, click icon adjacent to the field.

·   This is mandatory.

 

Level Type

drop-down list

The different hierarchy levels at which the ULD stock details is to be monitored

 

Possible values:

·   Head-Quarters

·   Region

·   Country

·   Airport

·   Facility

·   Agent

 

·   Select this from the drop-down list.

·   To open the list, click the arrow in the field.

 

Note:

While setting up a ULD stock it is mandatory to select the value as ‘Airline’

O

Level Value

field

The code of the entity corresponding to the level type.

 

The value in this changes based on the level type selected

 

·   Type this in the field or select from the LOV.

·   To open the LOV, click icon adjacent to the field

·   This is mandatory when Level Type is selected

C

Station Code

field

Code of the station, stock details of which are to be viewed

 

Type this in the field.

 

NOTE:

This field appears only when the Level Type is selected as ‘Agent’ or when blank is selected in Level Type.

 

O

·   Type this in the field or select from the LOV.

·   To open the LOV, click icon adjacent to the field.

O

·   Type this in the field or select from the LOV.

·   To open the LOV, click icon adjacent to the field.
