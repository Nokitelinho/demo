To access the SCM ULD Error Log screen:
Select the ULD Management tab. On the ULD Messages menu, click SCM ULD Error Logs.

 

When you access the screen, the logged in station is displayed in the Airport field.

 

The following users can use the SCM ULD Error Log screen.

·        Ground Handling Staff

·        Airline Staff

View list of SCM error logs

 

1.      Select the ULD Management tab. On the ULD Messages menu click SCM ULD Error Logs to get the SCM ULD Error Log screen.

 

2.      In the SCM ULD Error Log screen, specify the search criteria. The following are mandatory :

·        Type or select the Airline

·        Type or select the SCM Seq No. (SCM Sequence Number)

Note:

The Stock Chk Date (stock check date) and Stock Chk Time (stock check time) corresponding to the SCM Seq no (SCM sequence number) selected from the LOV gets displayed in the respective fields.

 

3.      Click List. (mandatory)

The SCM error logs for the specified search criteria appear in the table.

 

4.      To exit the screen, click Close.

Field/Button Description-SCM ULD Error Log screen

 

Reconcile ULD error-ULD not in airline stock

 

1.      Select the ULD Management tab. On the ULD Messages menu, click SCM ULD Error logs to get the SCM ULD Error Log screen.

 

2.      In the SCM ULD Error Log screen, list the SCM ULD errors. (mandatory)

 

3.      In the table, select the message with Error Description as ‘ULD not in Airline Stock’. (mandatory)

 

4.      Click Reconcile. (mandatory)

The Loan/Borrow Details screen appears.

 

5.      In the Loan/Borrow Details screen, specify the required details. (mandatory)

Field/Button Description-Loan/Borrow Details screen

 

6.      Click Save .(mandatory)

The error is reconciled and the error message disappears from the table in the SCM ULD Error Log screen.

 

7.      To exit from the screen, click Close.

Field/Button Description-SCM ULD Error Log screen

Reconcile ULD error- ULD not in station stock

 

1.      Select the ULD Management tab. On the ULD Messages menu, click SCM ULD Error Logs to get the SCM ULD Error Log screen.

 

2.      In the SCM ULD Error Log screen, list the SCM ULD errors. (mandatory)

 

3.      In the table, select the message with Error Description as ‘ ULD is not in station stock’. (mandatory)

 

4.      Click Reconcile. (mandatory)

The Record ULD Movement screen appears.

 

5.      In the Record ULD Movement screen, specify the required details. (mandatory)

Field/Button Description-Record ULD Movement screen

Note:

This is just a dummy movement for this ULD, so there is no need to specify the Flight No and Flight Date.

 

6.      Click Save. (mandatory)

The SCM ULD Error Log screen reappears with the updated information.

 

7.      To exit from the screen, click Close.

Field/Button Description-SCM ULD Error Log screen

Reconcile ULD error-ULD not in the system

 

1.      Select the ULD Management tab. On the ULD Messages menu, click SCM ULD Error Logs to get the SCM ULD Error Log screen.

 

2.      In the SCM ULD Error Log screen, list the SCM ULD errors. (mandatory)

 

3.      In the table, select the message with Error Description-‘ULD not in the system’. (mandatory)

 

4.      Click Reconcile. (mandatory)

The Maintain ULD screen appears.

 

5.      In the Maintain ULD screen, specify the required details. (mandatory)

Field/Button Description-Maintain ULD screen

 

6.      Click Save. (mandatory)

The error will be reconciled and the error message disappears from the table in the SCM ULD Error Log screen.

 

7.      To exit from the screen, click Close.

Field/Button Description-SCM ULD Error Log screen

Delete a ULD error message from system

 

1.      Select the ULD Management tab. On the ULD Messages menu click SCM ULD Error Logs to get the SCM ULD Error Log screen.

 

2.      In the SCM ULD Error Log screen, list the SCM ULD errors. (mandatory)

 

3.      In the table, select the row corresponding to the error message to be deleted. (mandatory)

 

4.      Click Delete.

Click Yes in the confirmation message that appears.

The error message will be permanently deleted from the system.

 

5.      To exit the screen, click Close.
