ULD MANAGEMENT

SCM VALIDATION

(Screen ID – ULD043)

Click here to view screen access information.

To access the SCM Validation screen:
Select the ULD Management tab. On the ULD Messages menu, click SCM Validation.

 

The following users can use the SCM Validationscreen:

·        ULD Controller

·        GHA staff

Functions

View SCM validation details

Print SCM Validation details

View SCM validation details

 

1.      Select the ULD Management tab. On the ULD Messages menu, click SCM Validation to get the SCM Validationscreen.

 

2.      In the SCM Validation screen, type or select the Airport. (mandatory)

 

3.      Click List. (mandatory)

The SCM validation details appear in the table.

 

4.      To exit the screen, click Close.

Field/Button Description – SCM Validation Screen

 

Print SCM Validation details

 

1.      Select the ULD Management tab. On the ULD Messages menu, click SCM Validation to get the SCM Validationscreen.

 

2.      In the SCM Validation screen, type or select the Airport. (mandatory)

 

3.      Click List. (mandatory)

The SCM validation details appear in the table.

 

4.      Click Print. (mandatory)

The SCM validation report is generated and printed.

 

5.      To exit the screen, click Close.

Facility Type

The facility type of the ULDs whose details are to be viewed.

 

Possible values:

·   AGENT LOC

·   CARGO

·   OPENSPACE

·   RAMP

·   REPAIR DOC

·   ULDDOC

·   WAREHOUSE

·   YARD

Location

field

The location corresponding to the ULDs whose details are to be viewed.
