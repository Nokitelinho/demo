Click here to view screen access information.

To access the Airport Facility Master screen:
Select the ULD Management tab. On the Stock Management menu, click Airport Facility Master.

 

When you access the screen, ‘Openspace’ is displayed as the default value in the Facility Type field.

 

The ULD Controller in the airline network can use the Airport Facility Master screen.

Create an airport facility

 

1.      Select the ULD Management tab. On the Stock Management menu, click Airport Facility Master to get the Airport Facility Master screen.

 

2.      In the Airport Facility Master screen, type or select the Airport Code (mandatory).

 

3.      Click List. (mandatory)

If the airport facility already exists, its details appear on the screen.

 

4.      Click the Add link. (mandatory)

A new row is added in the table.

 

5.      In the new row, specify the required details. The following are mandatory:

·        Select the Facility Code.

·        Type or select the Description.

 

6.      You can add more airport facilities at the same time. To do this click the Add link and repeat the above steps.

Note:

You can set the default location for a ULD by selecting the Default Flag check box. If the check box is selected, it indicates that the default location for a ULD will be corresponding to that particular facility code.

 

7.      Click Save. (mandatory)

The specified airport facilities are saved.

 

8.      To exit the screen, click Close.

View list of airport facilities

 

1.      Select the ULD Management tab. On the Stock Management menu, click Airport Facility Master to get the Airport Facility Master screen.

 

2.      In the Airport Facility Master screen, type or select the Airport Code. (mandatory)

 

3.      Click List. (mandatory)

The list of airport facilities appears in the table.

 

4.      To exit the screen, click Close.

Field/Button Description – Airport Facility Master screen

 

Modify an airport facility

 

1.      Select the ULD Management tab. On the Stock Management menu, click Airport Facility Master to get the Airport Facility Master screen.

 

2.      In the Airport Facility Master screen, type or select the Airport Code. (mandatory)

 

3.      Click List. (mandatory)

The list of airport facilities appears in the table.

 

4.      In the table, modify the details of the airport facility in the required row. (mandatory)

Note:

·        You can modify multiple airport facilities at a time. To do this, make the required changes in all corresponding rows.

·        All the fields are editable.

 

5.      Click Save. (mandatory)

The modifications are saved.

 

6.      To exit the screen, click Close.

Field/Button Description – Airport Facility Master screen

 

Delete an airport facility from the system

 

1.      Select the ULD Management tab. On the Stock Management menu, click Airport Facility Master to get the Airport Facility Master screen.

 

2.      In the Airport Facility Master screen, type or select the Airport Code. (mandatory)

 

3.      Click List. (mandatory)

The list of airport facilities appears in the table.

 

4.      In the table, select the required row. (mandatory)

Note:

You can delete multiple facilities at a time. To do this, select all the corresponding rows.

 

5.      Click the Delete link. (mandatory)

 

6.      Click Save. (mandatory)

The selected airport facilities are deleted from the system.

 

7.      To exit the screen, click Close.
