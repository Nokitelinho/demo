The System Administrator can use Maintain Service screen to create services.

 

Services are the additional facilities that are provided by the airlines for customers. These services include special handling of the cargo goods, door to door delivery of cargo goods and so on. A number of such services that are offered can be attached with a product or shown as a value-added service.

 

When you access the screen, default values are displayed in the following fields:

·             Status – ‘Active’

·             Service Duration – ‘0.0 Hours’

Create a service

 

1.      Open the Maintain Servicescreen ( Airline Functions > Masters > Products and Services > Maintain Service) .

 

2.      In this screen, specify general details of the service. The following are mandatory:

·          Service Code

·          Service Name

·          Description

·          Service Type

·          Transport Plan Display

·          Status

·          Service Duration

 

3.      Click Save. (mandatory)

 

A new service is created.

 

4.      To exit the screen, click Close.

Field/Button Description – Maintain Service screen

 

Create a service with restrictions

 

1.      Open the Maintain Servicescreen ( Airline Functions > Masters > Products and Services > Maintain Service) .

 

2.      In this screen, specify general details of the service. The following are mandatory:

·          Service Code

·          Service Name

·          Description

·          Service Type

·          Transport Plan Display

·          Status

·          Service Duration

 

3.      In the Restrictions section, click Add. (mandatory)

A new row appears in the Import stations/Export stations/ Commodity table. Specify the restriction details.

Note:

  • Selecting the Restrict radio button implies that the service defined is not applicable to the import stations/ export stations/ commodity specified.
  • Selecting the Allow radio button implies that the service defined is applicable only to the import stations/ export stations/ commodity specified

 

4.      Click Save. (mandatory)

A new service with restrictions is created.

 

5.      To exit the screen, click Close.

Field/Button Description – Maintain Service screen

 

Modify a service

 

1.      Open the Maintain Servicescreen ( Airline Functions > Masters > Products and Services > Maintain Service) .

 

2.      In this screen, type or select the Service Code of the service to be modified. (mandatory)

 

3.      Click List. (mandatory)

The details of the specified service appear.

 

4.      Make the required modifications.

Note:
Status and Service Code cannot be modified.

 

5.      If required, in the Restrictions section, you can modify the Import Station, Export Station or Commodity details by deleting the existing row and adding a new one. For this:

·          Select the required row and click Delete.

·          Click Add to add a new row and then specify the new values.

 

6.      Click Save. (mandatory)

The modified service is saved with the same service code.

 

7.      To exit the screen, click Close.

Field/Button Description – Maintain Service screen

 

Define time attributes for a service

 

1.      Open the Maintain Servicescreen ( Airline Functions > Masters > Products and Services > Maintain Service) .

 

2.      In this screen, specify general details of the service. The following are mandatory:

·          Service Code

·          Service Name

·          Description

·          Service Type

·          Transport Plan Display

·          Status

·          Service Duration

 

3.      Click Service Time Attributes. (mandatory)

Note:
The screen that pops up on clicking Service Time Attributes would be different, depending upon the service type specified. If the service type selected is Both/Export/Import then the Time Attributes - Export/Import screen is displayed. If the selected service type is recurring, then Time Attributes – Recurring screen is displayed.

 

4.      Enter the required details in the Time Attributes - Export/Import OR Time Attributes – Recurring screen.

Field/Button Description – Time Attributes- Export/Import screen

Field/Button Description – Time Attributes – Recurring screen

 

5.      Click OK. (mandatory)

The specified time attributes are saved

 

6.      To exit the screen, click Close.

M

Service Type
drop-down list

The type of the service created.

Possible values:

·          Both

The service is to be provided at the export stations and the import stations.

 

·          Export

The service is to be provided at the export station only.

 

·          Import

The service is to be provided at the import station only.

 

·          Recurring

The service is to be provided repeatedly during transit from the export station to the import station.

 

·       Select this from drop-down list.

 

·       To open the list, click the arrow in the field.

 

O

Transport Plan Display
drop-down list

The transportation plan (of the shipment) in which the service details are to be displayed.

 

Possible values:

·          Both

The service is to be displayed in both the external and internal transportation plans of the shipment.

·          External

The service is to be displayed only in the external transportation plan of the shipment.

·          Internal

The service is to be displayed only in the internal transportation plan of the shipment.

·          None

The service is not to be displayed in any transportation plan.

 

NOTE:

For the service details to be displayed in the transportation plan, the service is to be linked to a milestone.

 

·       Select this from drop-down list.

 

·       To open the list, click the arrow in the field.

 

O

Possible values:

·          Active

·          Inactive

 

By default, when a service is created, its status is ‘Active’. This cannot be edited.
