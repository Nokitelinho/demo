maintain sub product

Click here to view screen access information.

This screen can be accessed from the following screens.

Maintain Products (screen ID – PDS003)
   - Specify the Name and click List.
   - Click List Sub Products.
   - In the List Sub Product screen: In the Sub Product Details table, select the required row and click Details.

List Product (screen ID – PDS003)
   - Specify the required search criteria and click List.
   - In the Product Details table, select the required row and click List Sub Products.
   - In the List Sub Product screen: In the Sub Product Details table, select the required row and click Details.

 

description of Screen elements

Screen elements (fields, buttons, links and so on) are sequenced in the table as per the ‘tab order’ on the screen. (Tab order is the sequence in which the focus changes when you press the Tab key. This is usually left to right within each row of screen elements).

Milestone field

The mile stone to be covered by the product

 

This is a display field and cannot be edited.

D

Type
 field

The various type of the milestones are

·           Export – Milestones that occur before departure of the flight

·           Import – Milestones that occur after arrival of the flight

 

This is a display field and cannot be edited.

D

Minimum Time( Hrs)
field

·          For export milestones:

The time in hours within which the milestone can occur before departure

 

·          For import milestones:

The least time after which only the milestone should occur after arrival

 

Type this in the field.

 

M

Maximum Time( Hrs)
field

·          For export milestones:

The least time in hours before which the milestone should happen before departure

 

·          For import milestones:

The time within which the milestone should be completed after arrival.

 

Type this in the field.
