Time Attributes – Export/Import Screen

Click here to view screen access information.

This screen is accessed from the Maintain Service (screen ID – SHR026) screen.

- Specify the Service Type and click Service Time Attributes .

 

The Time Attributes - Export/Import screen is used to maintain or modify the service time details for export services, import services and services to be provided at the export and import side.

DESCRIPTION OF SCREEN ELEMENTS

Screen elements (fields, buttons, links and so on) are sequenced in the table as per the ‘tab order’ on the screen. (Tab order is the sequence in which the focus changes when you press the Tab key. This is usually left to right within each row of screen elements).
