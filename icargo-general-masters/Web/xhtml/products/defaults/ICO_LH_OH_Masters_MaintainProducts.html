Note:

To view/modify product details:

·        If you are not aware of the Product Code or Product Name of the product, you can use the List Products screen. In this screen, provide your search criteria, locate the required product and then view/modify its details.

Locate a product and view details

 

·        However, if you know the Product Code or Product Name of the product, you can use the Maintain Products screen. Specify the Product Code or Product Name of the product, and then display its details for viewing/modification.

 

Create a product

 

1.      Open the Maintain Products screen . ( Airline Functions > Masters > Products and Services > Maintain Products)

 

2.      In this screen, specify any one of the following details (mandatory)

·        Type the Product Code

·        Type the Product Name .

 

3.      Click Display Details. (mandatory)

·        If the product already exists in the system, its details appear on the screen.

·        If the product does not exist, a message appears enquiring if you wish to create a new one. Click Yes in the confirmation message, and proceed with the following steps.

 

4.      In the General tab, specify the details of the product. The following are mandatory:

·        Type the Description.

·        Type or select the To Date .

·        Select the Transport Mode .

·        Select the Priority .

·        Select th e SCC.

 

5.      To create time specification for milestones: (optional)

 

6.      In the Milestones tab: In the Time Specifications table, click the Add link . (mandatory)

The Milestone screen appears.

 

7.      In the Milestone screen that appears, select the required milestone. (mandatory)

 

8.      Click OK. (mandatory)

The selected milestone appears as a new row in the Time Specification table.

a)    The values in the following fields appear by default and can be edited, if required.

·        Minimum Time(Hrs)

·        Maximum Time(Hrs)

·        Alert( Minutes)

·        Chaser( Minutes)

·        Chaser Frequency

·        Maximum No Of Chasers

b)    Select any one of External, Internal or Transit checkboxes. (mandatory)

 

9.      To create restrictions for the product: (optional)

In the Restriction tab:

a)    To add restrictions for Commodity, Origin, Destination, Customer Group and Customer:( optional)

-   Select either Restrict or Allow option in the respective table to indicate if the entity is restricted or allowed for the product.

-   Click the Add link at the top right corner of the respective table, select the required items from the pop up screen that appears and click Ok.

b)    To add restrictions for Segment :(optional)

-   Select either Restrict or Allow option in the Routing Restriction table to indicate if the entity is restricted or allowed for the product.

-   Click the Add link. A new row appears in the table. You can specify the segment here.

c)    To restrict payment terms or mode of payments, select the required values from the Restricted Payment Terms or Mode Of Payment tables, respectively. Restricted types cannot be used for paying for this product. (optional)

d)    To add capacity restriction, specify the minimum and maximum weight and volume in the Capacity Restriction section. This ensures that the product will be provided only if shipment is of at least the minimum quantity and it will be available only up to the maximum quantity. (optional)

 

10.   Click Save. (mandatory)

Click Yes in the confirmation message that appears. Product Code is automatically generated and the product is saved

 

11.   To exit the screen, click Close.
