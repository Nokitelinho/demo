Click here to view screen access information.

To access the Allocate New Stock screen:
Select the Airline Functions tab. On the Stock Control menu, go to Stock and click Allocate New.

 

When you access the screen, default values are displayed in the following fields:

·        Doc Type. - default value set in the system

·        Sub Type – default value set in the system

 

The stock control department can use the Allocate New Stock screen.

 

Function

Allocate stock to a stock holder without request being raised

 

1.      Select the Airline Functions tab. On the Stock Control menu, go to Stock and click Allocate New to get the Allocate New Stock screen.

 

2.      In the Allocate New Stock screen, specify the search criteria. The following are mandatory:

·        Select the Doc Type.

·        Select the Sub Type .

·        Type or select the Approver .

·        Type or select the Stock Holder Code .

 

3.      You can list the available stocks and assign it to the right side table. (optional)

To do this:

a)    In the Available Stock section, specify the details. The following fields are mandatory:

·        Type Range From.

·        Type Range To or No Of Docs.(no of documents)

b)    Click List. (mandatory)

The matching stock ranges for the criteria specified are displayed in the Available Stock section. The total number of documents available for allocation is displayed in the Total no of Docs field.

c)    Select the range to be allocated and click >> to move it to the table on the right side table.

 

4.      In the right table, specify the Range From and Range To values of the desired stock range to be allocated.

Note:

·          Click the Add link to add different ranges of stock.

·          The total number of stock that has been allocated is displayed in the Total no of Docs field.

·          Only the stock in the available stock range can be allocated.

 

5.      To allocate the stock, click Save . (mandatory)

Click Yes in the confirmation message that appears.

The specified range is allocated to the specified stockholder.

 

6.      To exit the screen, click Close.
