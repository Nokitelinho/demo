View list of blacklisted stocks

 

1.      Select the Airline Functions tab. On the Stock Control menu, click List Blacklisted Stock to get the List BlackListed Stock screen.

 

2.      In the List BlackListed Stock screen, specify the search criteria. The following are mandatory:

·        Select the Doc Type.

·        Select the Sub Type.

·        Type or select the From Date.

·        Type or select the To Date.

 

3.      Click List. (mandatory)

The list of blacklisted stocks that match the specified search criteria appears in the Blacklisted Stock Details table.

 

4.      To exit the screen, click Close.

Field/Button Description – List BlackListed Stock screen

 

Revoke blacklist on stock

 

1.      Select the Airline Functions tab. On the Stock Control menu, click List Blacklisted Stock to get the List BlackListed Stock screen.

 

2.      In the List BlackListed Stock screen, list the blacklisted stocks. (mandatory)

 

3.      In the Blacklisted Stock Details table, select a row. (mandatory)

 

4.      Click Revoke. (mandatory)

The Revoke BlackListed Stock screen appears with the details of the blacklisted stock.

 

5.      In the Revoke BlackListed Stock screen.

a)    You can revoke:

i)     The entire stock range

ii)    A particular range of the stock

Specify the details. The following are mandatory.

-   Type the Range From.

-   Type the Range To.

Note:

To discontinue the process and return to the List BlackListed Stock screen, click Close.

b)    Click Revoke. (mandatory)

Click Yes in the confirmation message that appears.

The blacklist against the entire stock or a part of it is revoked.

The List BlackListed Stock screen appears with the updated range.

Field/Button Description – Revoke Blacklisted Stock screen

 

6.      To exit the screen, click Close.
