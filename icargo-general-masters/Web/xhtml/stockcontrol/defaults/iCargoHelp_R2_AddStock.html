EXPORT

Add Stock Screen

To access the Add Stock screen:
-           Select the Cargo Terminal Opr tab.
-           On the Export menu, go to AWB Stock andclick Stock Manager.
-           In the Stock Manager screen that appears, click List to enable the Add button.

-           Click Add to add the stock in the Add Stock screen.

 

This screen is accessible only if there are any results from the search or if u wants to add new stock.

 

Add Stock screen used by the stock controller to add the stock details of an airline.

 

In field descriptions, fields may be marked as Mandatory, Optional or Display Field.

 

Mandatory - Fields in which data entry is mandatory

Optional - Fields in which data entry is optional

Display Field - Non-editable fields in which data appears automatically

 

 

FIELD DESCRIPTION

 

STOCK DETAILS SECTION

 

Range From (Mandatory)

Starting number of the range to be added

Specify the range.

 

Range To (Mandatory)

Ending Number of the Range to be added

Specify the range.

 

Doc Type (Display Field)

Name of document to be created

 

Doc Sub Type (Display Field)

Document Type of document to be created

 

Remarks (Optional)

Remarks about the document

Specify this in the field.

 

THE TABLE

 

S No (Display Field)

Serial Number of the stock

Content (Display Field)

The content is displayed

 

Action Check box (Display Field)

Check if the request is to be actioned

Note
If the check box is checked, the request is actioned.

Only unactioned requests will be shown in this table.

If a request is checked to be actioned, it is actioned by the stock that is currently being made.

 

BUTTON DESCRIPTION

 

OK

Click Ok to save the details entered and return to the Stock Manager screen.

 

Cancel

Click Cancel to return to the Stock Manager screen without saving the details entered.
