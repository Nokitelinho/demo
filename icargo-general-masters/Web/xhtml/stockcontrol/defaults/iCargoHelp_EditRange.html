STOCK CONTROL

Edit Range Screen

 


To access the Edit Range screen:
-           Select the Airline Functions tab. On the Stock Control menu, go to Stock and click Allocate to get the Allocate Stock screen.
-           In the Allocate Stock screen, specify the Doc Type, Sub Type, Stock Control For, Status, From Date and To Date and click List.
-           Select the request for which allocation is to be done and click List in the Range section.
-           Click Edit Range

 

The Edit Range screen can be used by the Cargo Headquarters’ Staff, Stock Control Manager, Regional Cargo Manager and Cargo Station Manager to edit the range of stocks that are to be allocated to a particular stockholder, against a request made.

 

 

Related Topics

Create stock in the Create Stock screen

Allocate stock to a stockholder against a stockholder request in the Allocate Stock screen

 

 


In field descriptions, fields may be marked as Mandatory, Optional or Display Field.

 

Mandatory - Fields in which data entry is mandatory

Optional - Fields in which data entry is optional

Display Field - Non-editable fields in which data appears automatically

 

 

FIELD DESCRIPTION

 

Doc Type (Display Field)

The type of the document to be allocated as requested in the Maintain Stock Request screen

 

Sub Type (Display Field)

The sub type of the document to be allocated as requested in the Maintain Stock Request screen

 

Approved Stock (Display Field)

The number of approved stocks in the request selected in the Allocate Stock screen

 

Manual check box (Display Field)

This check box will be selected if the request selected in the Allocate Stock screen was made for manual stock

 

Stock Holder Code (Display Field)

The code of the stock holder who raised the request in the Maintain Stock Request screen

 

AVAILBLE STOCK SECTION

 

Range From (Mandatory)

The number from which the document ranges are to be displayed

For example, if 0 is given in the Range From field, it means that the system has to search for and display all the available stock matching the specified search criteria starting from the number 0 of the stock range.

 

Range To (Optional)

The number till which the document ranges are to be displayed

For example, if 9999999 is given in the Range To field, it means that the system has to search for and display all the available stock matching the specified search criteria till the number 9999999 of the stock range.

 

No of Docs (Optional)

The number of documents available with the approver of the requested stock holder

 

Note:
The range available with the approver for allocation would be displayed in the table section by default. Stock can be allocated directly from this range without editing. If this displayed range is to be changed by searching for other ranges, then specifying either Range From and Range To; or Range From and No of Docs is mandatory.

 

THE TABLE (LEFT)

 

Range From (Display Field)

The starting number of the document range available with the approver

For example, if 0000010 is displayed in the Range From field, it means that the approver has got stocks for allocation with it for a range starting from 0000010.

 

Range To (Display Field)

The number of the document range till which the stock is available with the approver

For example, if 0000050 is displayed in the Range To field, it means that the approver has got stocks for allocation with it till the range 0000050.

 

No of Docs (Display Field)

The number of stocks in the range displayed

 

Total no of Docs

The total number of stocks displayed for allocation

 

THE TABLE (RIGHT)

 

Range From (Optional)

The starting number of the document range to be allocated

For example, if 0000010 is entered in the Range From field, it means that the stock range starting from 0000010 is going to be allocated.

 

Range To (Optional)

The number of the document range till which it would be allocated

For example, if 0000050 is entered in the Range To field, it means the stock range till 0000050 is going to be allocated.

 

No of Docs (Optional)

The number of stocks in the range displayed to be allocated

 

BUTTON/LINK DESCRIPTION

 

List

Specify the details and click this button to display the available stock.

 

Add l Delete link

Click the Add link to add a new row in the table. Select a row and click the Delete link to delete the row.

 

>> 

Select the ranges to be transferred (in the table on the left side) and click this button to move the selected ranges to the table on the right.

 

OK

Click this button to save the changes made in the screen and to return to the Allocate Stock screen.

 

Cancel

Click this button to exit the screen without saving the details.
