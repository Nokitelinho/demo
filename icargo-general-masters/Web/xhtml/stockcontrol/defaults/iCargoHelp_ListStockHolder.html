STOCK CONTROL

List Stock Holder
(Screen: STK006)

 


To access the List Stock Holder screen:
Select the Airline Functions tab. On the Stock Control menu , go to Stock Holder and click List.

 

The List Stock Holder screen can be used by Cargo Headquarters to view the list of stockholders. You can specify your search criteria to get a matching stockholder list.

 

This screen provides link to Maintain Stock Holder screen.

Functions

List Stock Holder – Description of Screen Elements

 


To access the List Stock Holder screen:
Select the Airline Functions tab. On the Stock Control menu , go to Stock Holder and click List.

Function Description – List Stock Holder screen

 

Description of Fields and Buttons

Fields and buttons are described separately under different headings. They are given in the same order as they appear on screen.

Field Description

Button Description

 

In field descriptions, fields may be marked as Mandatory, Optional or Display Field.

 

Mandatory - Fields in which data entry is mandatory

Optional - Fields in which data entry is optional

Display Field - Non-editable fields in which data appears automatically

 

FIELD DESCRIPTION

 

       SEARCH CRITERIA SECTION

 

Stock Holder Type (Mandatory)

The type of the stockholder whose details are to be listed

Select this from the drop-down list.

 

Stock Holder Code (Optional)

The code of the stockholder whose details are to be listed

Specify this in the field or select from the LOV.

Note:
The values in the Stockholder LOV are populated according to the value selected in the Stock Holder Type field.

 

Doc Type . (Optional)

The type of the document whose details are to be listed

Select this from the drop-down list.

 

Sub Type (Optional)

The sub type of the document whose details are to be listed

Select this from the drop-down list.

Note:
The values in the Sub Type drop-down list are populated according to the value selected in the Doc Type field.
If more than one sub type for a doc type exists, then select the required sub type from the drop-down list.

 

       STOCK HOLDER DETAILS SECTION

 

Stock Holder

The details of the stockholder is displayed

 

Type (Display Field)

The type of the stockholder

This can be Headquarters, Region, Station or Agent.

 

Code (Display Field)

The code of the stockholder

The value corresponding to the stockholder type is displayed.

 

Doc Type ( Display Field)

The type of the document allocated to the displayed stockholder

 

Sub Type (Display Field)

The sub type of the document allocated to the displayed stockholder

 

Reorder Level (Display Field)

The number of stocks that should be with the stock holder, such that if the number of stocks with the stock holder becomes equal to the specified value or falls below this level, a request should be raised.

 

Reorder Quantity (Display Field)

The quantity of documents for which the request has to be raised, in case of a reorder

 

Reorder Alert check box(Display Field)

If this check box is selected, it indicates that an alert would be automatically sent  to the specified contact id when the stock level reaches or falls below the reorder level.

 

Auto Stock Request check box(Display Field)

If this check box is selected, it indicates that a request would be automatically sent for the reorder quantity set in the system, when the stock level reaches or falls below the reorder level.

 

BUTTON DESCRIPTION

 

List

Specify search criteria and click this button to get a list of stockholders matching the search criteria.

 

Clear

Click this button to clear all entries simultaneously.

 

Monitor

Select a row and click this button to view the distribution details of the selected stock range in the Monitor Stock screen.

 

Create

Click this button to navigate to the Maintain Stock Holder screen, where stockholders can be created.

 

Details

Select a row and click this button to view/modify the details of the corresponding stockholder.

 

Delete

Select a row and click this button to delete the selected stockholder.

 

Close

Click this button to exit the screen.
