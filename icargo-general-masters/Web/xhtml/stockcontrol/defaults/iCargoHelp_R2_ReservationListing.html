Reservation Listing
(Screen: STK010)

To access the Reservation Listing screen:
Select the Cargo Terminal Opr tab. On the Export menu, go to AWB and click List AWB Reservation .

 

Reservation Listing screen is used by Airline handling staff or registered agents to view the list of reserved stocks. You can specify your search criteria to get a list of reserved AWB’s.

 

You can search for a reservation from a particular Reservation Date to another Reservation Date or from a certain Expiry Date to another Expiry Date.

View list of reservation details

 

1.        Select the Cargo Terminal Opr tab. On the Export menu, go to AWB and click List AWB Reservation to get the Reservation Listing screen.

 

2.        In this screen, specify the required search criteria. The following fields are mandatory:

-           From date and To date of the Reservation Date

OR

-           From date and To date of the Expiry Date

 

3.       Click List .

The list of AWB’s that are reserved will be shown. Details such as the AWB number, AWB Type, Cust. Code, Reservation Date, Expiry Date Remarks and so on will be shown.

Note:
Click Clear to clear all search criteria entries simultaneously.

 

4.        Click Close to exit the screen

 

Field/Button Description – Reservation Listing screen

 

Extend the period of reservation for an AWB

Reservation of an AWB can be extended on request by a shipper to the Ground handling Agent.

 

1.        Select the Cargo Terminal Opr tab. On the Export menu, go to AWB and click List AWB Reservation to get the Reservation Listing screen .

 

2.        In this screen, specify the search criteria. The following fields are mandatory:

-           From date and To date of the Reservation Date

OR

-           From date and To date of the Expiry

 

3.        Click List.

The details of the reserved AWBs matching the search criteria will be listed.

Note:
Click Clear to clear all search criteria entries simultaneously.

 

4.        Select the AWB(s) whose reservation period has to be extended and click Extend Reservation.

The Extend Reservation screen appears. Here you can change the Expiry Date of the AWB.

Note:
The Expiry Date should not be greater than the Maximum Amend Date

If more than one AWB is selected, then click the First|Next|Prev|Last links to navigate to other AWB’s.

Field/Button Description – Extend Reservation Screen

 

5.        Click Ok to save the details of the AWB Expiry Date and return to the Reservation Listing screen.

 

6.        Click Close to exit the Reservation Listing screen.

Reservation Listing – Description of Screen Elements

 

To access the Reservation Listing screen:
Select the Cargo Terminal Opr tab. On the Export menu, go to AWB and click List AWB Reservation.

Function Description – Reservation Listing Screen

 

Description of Fields and Buttons

Fields and buttons are described separately under different headings. They are given in the same order as they appear on screen.

Field Description

Button Description

 

In field descriptions, fields may be marked as Mandatory, Optional or Display Field.

 

Mandatory - Fields in which data entry is mandatory

Optional - Fields in which data entry is optional

Display Field - Non-editable fields in which data appears automatically

 

FIELD DESCRIPTION

 

SEARCH CRITERIA SECTION

 

Reservation Date:

From , To (Mandatory)

The start date and end date of the reservation within which reserved AWBs were checked for

Specify these in the field or select it from the calendar to search for AWBs reserved during the specified period.

 

Airline (Optional)

The airline who’s AWB was reserved

Specify this in the field to narrow down your search for a particular reserved AWB.

 

Cust Code (Optional)

The code of the customer for whom the AWB is reserved

Specify this in the field.

 

AWB Type (Optional)

The drop-down list contains the different types of AWB’s.

The possible values are: Manual AWB’s and System AWB’s.

-           Manual AWB’s are those that the Ground Handling agent has with him in hand. i.e. a hard copy, of which he will have to enter all the necessary details.

-           System AWB’s are those wherein the system generates the AWB on the spot. The Ground Handling agent fills in the necessary details and then orders a print to be given to the shipper.

 

Select an AWB of a particular type from the drop-down list in order to search for reserved AWB’s.

 

Expiry Date :

From , To (Mandatory)

The start date and end date of the expiry date within which reserved AWB’s were checked for

Specify these in the field or select it from the calendar to search for reserved AWB’s having expiry date within the period specified.

 

THE TABLE SECTION

 

AWB (Display Field)

The unique Air Way bill number on which a shipment travels

The reserved AWB matching the search criteria

The first part of the field displays the airline prefix in and the second part of the field displays the AWB number of up to eight alphanumeric characters

 

Type (Display Field)

The type of AWB reserved

 

Cust. Code (Display Field)

The code of the customer for whom the AWB was reserved

 

Res. Date (Display Field)

The date on which the AWB was reserved

 

Expiry Date (Display Field)

The date on which the reservation of the AWB expires

 

Remarks (Display Field)

The remarks on the reservation of the AWB

 

BUTTON DESCRIPTION

 

List

Specify search criteria and click this button to get the list of reserved AWB’s matching the search criteria.

 

Clear

Click this button to clear all entries simultaneously.

 

Extend Reservation

Select the AWB(s) and click this button to extend the reservation period of the AWB.

 

Cancel Reservation

Select the AWB(s) and click this button to cancel a reservation of the AWB.

 

View

Click this button to have a preview of the screen that is going to be printed.

 

Print

Click this button to print the list of reserved AWB’s.

 

Close

Click this button to exit the screen.
