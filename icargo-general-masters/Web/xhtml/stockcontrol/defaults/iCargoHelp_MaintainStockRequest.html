STOCK CONTROL

Maintain Stock Request
(Screen: STK015)

 


To access the Maintain Stock Request screen:
Select the Airline Functions tab. On the Stock Control menu, go to Stock Request and click Maintain.

 

The Maintain Stock Request screen is used by the Cargo Headquarters’ Staff, Stock Control Manager, Regional Cargo Manager, Cargo Station Manager and General Sales Agents to raise stock request to their approver. This request can be approved or rejected by the respective approver, according to their discretion.

Raise a stock request

1.        Select the Airline Functions tab. On the Stock Control menu, go to Stock Request and click Maintain to get the Maintain Stock Request screen.

 

2.        In the Maintain Stock Requestscreen, specify general details of the stock request. The following are mandatory:

-           Date of Req

-           Doc Type

-           Sub Type

-           Stock Holder Type

-           Stock Holder Code

-           Req. Qty

Note:
-     The values in the Stock Holder Code LOV are populated according to the value selected in the Stock Holder Type drop-down list.
-     If manual stock is requested, select the Manual check box.

 

3.        Click Save.

A new stock request is created with a unique reference number.

Maintain Stock Request– Description of Screen Elements   

 


To access the Maintain Stock Request screen:
Select the Airline Functions tab. On the Stock Control menu, go to Stock Request and click Maintain.

Function Description – Maintain Stock Request screen

 

Description of Fields and Buttons

Fields and buttons are described separately under different headings. They are given in the same order as they appear on screen.

Field Description

Button Description

 

In field descriptions, fields may be marked as Mandatory, Optional or Display Field.

 

Mandatory - Fields in which data entry is mandatory

Optional - Fields in which data entry is optional

Display Field - Non-editable fields in which data appears automatically

 

 

FIELD DESCRIPTION

 

Req Ref No (Display Field)

The system generated unique request reference number that identifies a stock request

 

Date of Req (Display Field)

The date on which the request is made

The current date would be populated here by default. This can be changed if required.

Specify this in the field or select from the calendar.

 

  Status (Display Field)

The status of the request - New, Completed, Approved, Rejected or Cancelled

 

Doc Type (Mandatory)

The type of document that is to be requested

Select this from the drop-down list

 

Sub Type (Mandatory)

The sub type of the document that is to be requested

Select this from the drop-down list

Note:
The values in the Sub Type drop-down list is populated according to the Doc Type selected

 

Manual check box (Optional)

Select this check box to request for printed stocks

 

Stock Holder Type (Mandatory)

The type of stock holder raising the request

Select this from the drop-down list

 

Stock Holder Code (Mandatory)

The code of the stock holder raising the request

Specify this in the field or select from the LOV

Note:
The values in the Stock Holder Code LOV is populated according to the Stock Holder Type selected

 

Req Qty (Mandatory)

The requested quantity or the number of stocks requested

Specify this in the field

 

Remarks (Optional)

Remarks, if any, on the embargo

Specify this in the field

 

Allocated Stock (Display Field)

The number of stocks allocated by the approver to the person who has raised the request

 

App/Rej Remarks (Display Field)

Remarks made, if any, by the approver while approving / rejecting the request

 

BUTTON DESCRIPTION

 

Save

Click this button to save the entries made.

 

Clear

Click this button to clear all entries simultaneously.

 

Close

Click this button to exit the screen.
