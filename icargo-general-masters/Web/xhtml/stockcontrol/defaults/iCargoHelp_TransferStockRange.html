STOCK CONTROL

Transfer Stock Range Screen

 


To access the Transfer Stock Range screen:
-     Select the Airline Functions tab. On the Stock Control menu, go to Stock and click Monitor to get the Monitor Stock screen.
-     In the Monitor Stock screen, specify the Stock Holder Code, Doc Type and Sub Type and click List.
-     Select a row and click Transfer to get the Transfer Stock Range screen.

 

The Transfer Stock Range screen can be used by Cargo Headquarters, Stock Control Manager, Regional Cargo Manager and Cargo Station Manager to transfer stock from one stockholder to another stockholder in the same level of hierarchy.

In field descriptions, fields may be marked as Mandatory, Optional or Display Field

 

Mandatory - Fields in which data entry is mandatory

Optional - Fields in which data entry is optional

Display Field - Non-editable fields in which data appears automatically

 

FIELD DESCRIPTION

 

Transfer From (Display Field)

The code of the stockholder from whom the stocks are to be transferred

 

Manual check box (Display Field)

This check box would be selected if printed stock or documents are selected for transfer

 

Doc Type (Display Field)

The type of the document to be transferred

 

Sub Type (Display Field)

The sub type of the document to be transferred

 

AVAILBLE STOCK RANGE SECTION

 

Range From (Display Field)

The starting number of the document range from which the stock is to be transferred

 

Range To (Display Field)

The number of the document range till which the stock is to be transferred

 

No of Docs (Display Field)

The number of stocks in the range displayed

 

Total Available Stock (Display Field)

The total number of stocks available with the stockholder for transfer

This would be the sum of the number of documents displayed in Available Stock Range section

 

Transfer To (Mandatory)

The code of the stock holder to which the stocks are to be transferred

Specify this in the field or select from the LOV

Note:
Transfer can be done only to a stockholder in the same level of hierarchy

 

By Number of Documents radio button (Optional)

Select this radio button if the stock is to be transferred by number of documents, and not by range

 

Range From (Optional)

The starting number of the document range from which the stock is to be transferred

 

Number of Documents (Optional)

The number of documents that are to be transferred, starting from the number specified in the Range From field

 

By Range radio button (Optional)

Select this radio button if the stock is to be transferred by range, and not by number of documents

 

RANGE TO BE TRANSFERED SECTION

 

Range From (Optional)

The starting number of the document range from which the stock is to be transferred

 

Range To (Optional)

The number of the document range till which the stock is to be transferred

 

No of Docs (Display Field)

The number of documents in the range specified to be transferred

 

Total no of Docs (Display Field)

The total number of documents selected to be transferred

 

Remarks (Optional)

Remarks on the transferred stock, if any

 

BUTTON/LINK DESCRIPTION

 

Add l Delete links

Click Add to add a new row in the table. Select a row and click Delete to delete the row.

 

Transfer

Specify the details and click this button to transfer the stock

 

Close

Click this button to exit the screen.
