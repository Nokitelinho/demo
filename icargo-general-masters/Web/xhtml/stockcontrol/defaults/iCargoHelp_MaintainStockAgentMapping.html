STOCK CONTROL

Maintain Stock Agent Mapping
(Screen: STK017)

 


To access the Maintain Stock Agent Mapping screen:
Select the Airline Functions tab. On the Stock Control menu, go to Stock Holder and click Stock Holder Agent Mapping.

 

The Maintain Stock Agent Mapping screen can be used by Cargo Headquarters to maintain a mapping between stockholders and agents.

Functions

       View the list of existing agent-stock holder mappings

       Map an agent to a stock holder

       Modify an existing agent-stock holder mapping

 

       Related Topics

Create a stock holder in the Maintain Stock Holder screen

Create an agent profile in the Maintain Agent screen

 

 


       View the list of existing agent-stock holder mappings

1.        Select the Airline Functions tab. On the Stock Control menu, go to Stock Holder and click Stock Holder Agent Mapping to get the Maintain Stock Agent Mapping screen.

 

2.        In the Maintain Stock Agent Mapping screen, specify the search details of the agent - stock holder mappings to be displayed in the search section.

 

3.        Click List.

The matching records with the search criteria specified are displayed

Note:
If all the existing mappings in the system are to be listed, click List without specifying any search criteria

 

4.        Click Close to exit the Maintain Stock Agent Mapping screen

 

Field/Button Description - Maintain Stock Agent Mapping screen

 

       Map an agent to a stock holder

 

1.        Select the Airline Functions tab. On the Stock Control menu, go to Stock Holder and click Stock Holder Agent Mapping to get the Maintain Stock Agent Mapping screen.

 

2.        Specify the Agent and Stock Holder details in the respective fields.

 

3.        Click Add to add new rows in case more than one mapping is to be done.

 

4.        Click Save.

The specified mappings are saved and the Maintain Stock Agent Mapping screen appears with the updated mapping details.

Note:
One agent can be mapped only to one stockholder while one stockholder can have more than one agent mappings

 

5.        Click Close to exit the Maintain Stock Agent Mapping screen

 

Field/Button Description - Maintain Stock Agent Mapping screen

 

       Modify an existing agent-stock holder mapping

 

1.        Select the Airline Functions tab. On the Stock Control menu, go to Stock Holder and click Stock Holder Agent Mapping to get the Maintain Stock Agent Mapping screen.

 

2.        In the Maintain Stock Agent Mapping screen, specify the search details of the agent - stock holder mappings to be displayed in the search section.

 

3.        Click List.

The matching records with the search criteria specified are displayed

Note:
If all the existing mappings in the system are to be listed, click List without specifying any search criteria

 

4.        Make the required modifications

 

5.        Click Save

The modifications are saved and the Maintain Stock Agent Mapping screen appears with the updated mapping details

Note:
One agent can be mapped only to one stock holder while one stock holder can have more than one agent mappings

 

6.        Click Close to exit the Maintain Stock Agent Mapping screen

Function Description – Maintain Stock Agent Mapping screen

 

Description of Fields and Buttons

Fields and buttons are described separately under different headings. They are given in the same order as they appear on screen.

Field Description

Button/Link Description

 

In field descriptions, fields may be marked as Mandatory, Optional or Display Field.

 

Mandatory - Fields in which data entry is mandatory

Optional - Fields in which data entry is optional

Display Field - Non-editable fields in which data appears automatically

 


FIELD DESCRIPTION

 

       SEARCH CRITERIA SECTION

 

Stock Holder (Optional)

The code of the stock holder whose mapping details are to be displayed

Specify this in the field or select from the LOV.

 

Agent (Optional)

The code of the agent whose mapping details are to be displayed

Specify this in the field or select from the LOV.

 

       THE TABLE SECTION

 

Agent (Mandatory)

The code of the agent that is being mapped with a stock holder

Specify this in the field or select from the LOV.

 

Stock Holder (Mandatory)

The code of the stock holder that is being mapped with an agent

Specify this in the field or select from the LOV.

 

BUTTON/LINK DESCRIPTION

 

List

Specify search criteria and click this button to get a list of mappings matching the search criteria.

 

Clear

Click this button to clear all entries simultaneously.

 

First l Next l Prev l Last links

Click these links to move to the first/next/previous/last page, in the table.

 

Add l Delete links

Click Add to add a new row in the table. Select a row and click Delete to delete the row.

 

Save

Click this button to save the entries made.

 

Close

Click this button to exit the screen.
