EXPORT

Remove Stock Screen

To access the Remove Stock screen:
-           Select the Cargo Terminal Opr tab.
-           On the Export menu, go to AWB Stock andclick Stock Manager.
-           In the Stock Manager screen that appears, give the search criteria and click List
-           Select a row from the list of stocks for a specified airline
-           Click Remove

This screen is accessible only if there are any results from the search.

 

Remove Stock screen used by the stock controller to remove the selected row of the stock from a specified airline and remove the stock in between the ranges

 

In field descriptions, fields may be marked as Mandatory, Optional or Display Field.

 

Mandatory - Fields in which data entry is mandatory

Optional - Fields in which data entry is optional

Display Field - Non-editable fields in which data appears automatically

 

 

FIELD DESCRIPTION

 

CURRENT RANGE DETAILS SECTION

 

Range From (Display Field)

Starting Number of range of document in stock

 

Range To (Display Field)

Last Number of range of document in stock

 

Doc Type (Display Field)

Name of document to be removed.

 

Doc Sub Type (Display Field)

Document Type of document to be removed

 

REMOVAL DETAILS SECTION

 

Range From (Mandatory)

Starting number of the range of be removed

Specify this in the field

Note
Mention the range of stock will be removed.

 

Range To (Mandatory)

Last number of the range to be removed.

Note
Mention the range of stock will be removed

 

Remarks (Optional)

Any remarks specify in this field

 

BUTTON/LINK DESCRIPTION

 

First/previous/next/last

 

Remove

Select a row and click this button to remove the selected stock details in the Remove Stock screen.

 

Cancel

Click Cancel to return to the Stock Manager screen without saving the details entered.
