Stock List
(Screen: STK009)

To access the Stock List screen:
Select the Cargo Terminal Opr tab. On the Export menu, go to AWB Stock andclick Stock List.

 

The Stock List screen is used by the Stock Controller to list the stock of the particular airline, view the request and create a request for an airline to be created and print the list of stocks in the screen.

View the list of stock

1.        Select the Cargo Terminal Opr tab. On the Export menu, go to AWB Stock and click Stock List to get the Stock List screen .

 

2.        In thisscreen, specify the required search criteria. The following are mandatory:

-           Doc Type

-           Doc Sub Type

 

3.        Click List.
The list of stocks that satisfy the search criteria gets displayed.

Note :
Click Clear to clear all the entries simultaneously

 

4.        Click Close to exit the screen.

Create request

 

1.        Select the Cargo Terminal Opr tab. On the Export menu, go to AWB Stock and click Stock List to get the Stock List screen .

 

2.        Click Create Request to create a request for an airline to be created in Create Request screen.

 

3.        In thisscreen, specify the required details. The following are mandatory:

-           Airline

-           Doc Type

-           Doc Sub Type

-           Address

 

4.        Click Ok to save the details entered and return to the Stock List screen.

 

5.        Click Close to exit the screen and return to the Stock List screen.

Stock List – Description of Screen Elements      

 


To access the Stock List screen:
Select the Cargo Terminal Opr tab. On the Export menu, go to AWB Stock and click Stock List

Function Description – Stock List screen

 

Description of Fields and Buttons

Fields and buttons are described separately under different headings. They are given in the same order as they appear on screen.

Field Description

Button/Link Description

 

In field descriptions, fields may be marked as Mandatory, Optional or Display Field.

 

Mandatory - Fields in which data entry is mandatory

Optional - Fields in which data entry is optional

Display Field - Non-editable fields in which data appears automatically

FIELD DESCRIPTION

 

SEARCH CRITERIA SECTION

 

Airline (Optional)

The airline whose stock details have to be viewed

Specify this in the field

 

Doc Type (Mandatory)

Type of the Document whose stock is to be listed

Select this from the drop down list.

Note
Doc.sub type will be changed corresponding to the Doc Type

 

Doc Sub Type (Mandatory)

Document sub type whose stock is to be listed

Select this from the drop down list.

Note
Doc.sub type will be changed corresponding to the Doc Type

 

STOCK LIST SECTION

 

Airline (Display Field)

Code of the airline owning the stock documents.

 

Doc Type (Display Field)

Type of the Document whose stock is to be listed

 

Doc Sub Type (Display Field)

Document sub type whose stock is to be listed

 

Total Stock (Display Field)

Total number of documents in the stock

 

Reorder Level (Display Field)

A preconfigured number for a particular airline, which prevents the total stock from falling below this level.

 

BUTTON DESCRIPTION

 

List

Specify the required search criteria and click this button to view the stock list.

 

Clear

Click this button to clear all entries simultaneously.

 

View Request

Click this button to view the request of the airline in the View Request screen.

 

Create Request

Click this button to create a request for an airline to be created in Create Request screen.

 

Stock Manager

Click this buttonto view or add /remove the stock details for the particular airline’s Document type and Document Sub Type in Stock Manager screen.

 

View

Click this buttonto view the report of the airline.

 

Print

Click this button to get the print of listed stock.

 

Close

Click this button to exit the screen.
