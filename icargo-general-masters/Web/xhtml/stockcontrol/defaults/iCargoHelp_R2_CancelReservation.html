Cancel Reservation Screen     

To access the Cancel Reservation screen:
-           Select the Cargo Terminal Opr tab.
-           On the Export menu, go to AWB andclick List AWB Reservation.
-           In the Reservation Listing screen that appears, specify the search criteria and click List.
-           From the list, s elect the reservations to be cancelled.

-           Click Cancel Reservation.

 

In the Reservation Listing screen, i f more than one reservation has been selected for cancellation, the AWB No. and Remarks of the first selected reservation is displayed in the Cancel Reservation screen.  You can use the First| Next| Previous| Last links to view details of the remaining reservations

 

Click OK in the Cancel Reservation screen to cancel all the reservations selected in the Reservation Listing screen

 

In field descriptions, fields may be marked as Mandatory, Optional or Display Field.

 

Mandatory - Fields in which data entry is mandatory

Optional - Fields in which data entry is optional

Display Field - Non-editable fields in which data appears automatically

 

FIELD DESCRIPTION

 

AWB (Display Field)

The unique bill number on which a shipment travels

 

Remarks (Optional Field)

The remark specified against the reservation in the Reservation Listing screen appears here.

This can be modified if required.

 

BUTTON/LINK DESCRIPTION

 

First|Next|Previous|Last links

Click these links to move to the first/next/previous/last page, in the table.

 

OK

Click this button to cancel all the selected reservations and return to Reservation Listing screen.

 

Close

Click this button to close the screen and return to the Reservation Listing screen.
