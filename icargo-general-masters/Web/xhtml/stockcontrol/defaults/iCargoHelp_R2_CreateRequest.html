EXPORT

Create Request Screen

To access the Create Request screen:
-           Select the Cargo Terminal Opr tab.
-           On the Export menu, go to AWB Stock andclick Stock List.
-           In the Stock List screen that appears, give the search criteria and click List
-           The list of stocks will be populated and shown in Stocklist table.
-           Select a row and click Create Request , the Create Request screen will open

 

The Create Request screen is used by stock controller to create stock request for the particular airline to be created .

 

Mandatory - Fields in which data entry is mandatory

Optional - Fields in which data entry is optional

 

FIELD DESCRIPTION

 

CREATE REQUEST SECTION

 

Airline (Mandatory)

Code of the Airline whose request is to be created

Specify this in the field.

 

Doc Type (Mandatory)

Type of document whose request is to be created

Select this from the drop down list

Note
Doc.sub type will be changed corresponding to the Doc Type

 

Doc Sub Type (Mandatory)

Type of sub document whose request is to be created

Select this from the drop down list

Note
Doc.sub type will be changed corresponding to the Doc Type

 

Mode Of Communication (Optional)

The mode of communication for the request is displayed

Select this from the drop down list

 

Address (Mandatory)

The Address to which communication is to be sent

Specify this in the field.

 

Content (Optional)

Content is of the request if any.

Specify this in the field.

 

BUTTON DESCRIPTION

 

OK

Click Ok to save the details entered and return to the Stock List screen.

 

Close

Click Close to return to the Stock List screen without saving the details entered.
