EXPORT

View Request Screen

To access the View Request screen:
-           Select the Cargo Terminal Opr tab.
-           On the Export menu, go to AWB Stock andclick Stock List.
-           In the Stock List screen that appears, give the search criteria and click List
-           The list of stocks will be populated and shown in a table.
-           Select a row and click View Request to get View Request screen


The View Request screen is used by the stock controller to view request of the stock for the particular airline

 

In field descriptions, fields may be marked as Mandatory, Optional or Display Field.

 

Mandatory - Fields in which data entry is mandatory

Optional - Fields in which data entry is optional

Display Field - Non-editable fields in which data appears automatically

 

 

FIELD DESCRIPTION

 

VIEW REQUEST SECTION

 

Airline (Display Field)

Code of the Airline whose request is to be listed

 

Doc type (Display Field)

The Document Type to be listed is displayed.

 

Doc Sub Type (Display Field)

The Document Sub Type to be listed is displayed

 

Mode Of Communication (Display Field)

The mode of communication for the request

 

Content (Display Field)

The content is displayed.

 

BUTTON DESCRIPTION

 

Close

Click Close to exit the screen and return to the Stock List screen.
