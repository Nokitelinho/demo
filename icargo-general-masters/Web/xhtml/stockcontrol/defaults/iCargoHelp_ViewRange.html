STOCK CONTROL

View Range Screen

 


To access the View Range screen :
-     Select the Airline Functions tab. On the Stock Control menu, go to Stock and click Monitor to get the Monitor Stock screen.
-     In thisscreen, specify Stock Holder Code , Doc Type and Sub Type and click List.
-     Select a row from the table and Click View Range .

 

The View Range screen displays details of the available stock range and the allocated stock range of a stockholder.

 

In field descriptions, fields may be marked as Mandatory, Optional or Display Field.

 

Mandatory - Fields in which data entry is mandatory

Optional   - Fields in which data entry is optional

Display Field - Non-editable fields in which data appears automatically

 

FIELD DESCRIPTION

Stock Holder ( Display Field )

The code of the stock holder

 

Doc Type (Display Field )

The Document Type of the stock

 

Sub Type ( Display Field )

The Sub Type of the stock

 

Manual ( Display Field )

This check box appears selected if the screen displays details of printed stock (manual/hard copy

documents)

 

AVAILABLE STOCK RANGE SECTION

 

Range From - Range To (Display Field)

The range of serial numbers of the available stock (i.e. the ‘From’ serial number and the ‘To’ serial number)

 

No. Of Doc (Display Field)

The number of documents available within a particular range

 

Total Available Stock (Display Field)

The total number of documents available

 

ALLOCATED STOCK RANGE SECTION

 

Range From - Range To (Display Field)

The range of serial numbers of the allocated stock (i.e. the ‘From’ serial number and the ‘To’ serial number)

 

No. Of Doc (Display Field)

The number of documents allocated for a particular range

 

Total Allocated Stock (Display Field)

The total number of documents allocated

BUTTON DESCRIPTION

Close

Click this button to exit thescreen.
