STOCK CONTROL

List Stock Holder

(Screen: STK006)

Click here to view screen access information.

To access the List Stock Holder screen:
Select the Airline Functions tab. On the Stock Control menu, go to Stock Holder and click List.

 

When you access the screen, default values are displayed in the following fields:

·        Stock Holder Type – ‘Headquarters’

·        Doc Type (Document Type) - ‘ AWB’

·        Sub Type – ‘S’

 

This screen provides link to Maintain Stock Holder, Monitor Stock screen

 

The following users can use the List Stock Holder screen:

·        Cargo Headquarters

·        Stock Control Manager

Functions

View list of stock details of a type of stock holder

Locate a stock holder and view/modify stock holder details

Locate a stock holder and monitor stock details of a stock holder type

Delete a stock holder

Note:

·        To create a new stock holder: Click Create in the List Stock Holder screen. This takes you to the Maintain Stock Holder screen, where you can create stock holders.

 

·        To view/modify stock holder details:

-          If you are not aware of the stock holder, you can use the List Stock Holder screen. In this screen, provide your search criteria, locate the required stock holder and then view/modify its details.

-          However, if you know the stock holder, you can use the Maintain Stock Holder screen. Specify the stock holder details, and then display its details for viewing/modification.

View list of stock holder details of a type of stock holder

 

1.      Select the Airline Functions tab. On the Stock Control menu, go to Stock Holder and click List to get the List Stock Holder screen.

 

2.      In the List Stock Holder screen: In the Search Criteria section, specify the required search criteria. The following are mandatory:

·          Select the Stock Holder Type

·          Select the Doc Type

·          Select the Sub Type.

 

3.      Click List. (mandatory)

The list of stock holders and its associated details appear in the screen.

 

4.      To exit the screen, click Close.

Field/Button Description – List Stock Holder screen

 

Locate a stock holder and view/modify stock holder details

If you are not aware of the stock holder, you can use the List Stock Holder screen. In this screen, provide your search criteria, locate the required stock holder and then view/modify its details.

 

Note:

However, if you know the stock holder, you can use the Maintain Stock Holder screen. Specify the stock holder details, and then display its details for viewing/modification.

View details of a stock holder

Modify details of a stockholder

 

1.      Select the Airline Functions tab. On the Stock Control menu, go to Stock Holder and click List to get the List Stock Holder screen.

 

2.      In the List Stock Holder screen, list the details of a stock holder for a stock holder type. (mandatory)

 

3.      In the Stock Holder Details table, select a row. (mandatory)

 

4.      Click Details. (mandatory)

The Maintain Stock Holder screen appears with the details of the selected stock holder.

 

5.      In the Maintain Stock Holder screen:

a)    Make the required changes.

The Stock Holder Type and Stock Holder Code are non-editable

Modify details of stock holder

Field/Button Description – Maintain Stock Holder screen

b)    Click Save. (mandatory)

The modifications are saved.

c)    Click Close. (mandatory)

The List Stock Holder screen reappears with the updated stock holder details.

 

6.      To exit the screen, click Close.

Field/Button Description – List Stock Holder screen

 

Locate a stock holder and monitor stock details of a stock holder type

 

1.      Select the Airline Functions tab. On the Stock Control menu, go to Stock Holder and click List to get the List Stock Holder screen.

 

2.      In the List Stock Holder screen, list the details of a stock holder for a stock holder type. (mandatory)

 

3.      In the Stock Holder Details table, select a row. (mandatory)

 

4.      Click Monitor.

The Monitor Stock screen appears where you can monitor the stock details of the stock holder.

 

5.      In the Monitor Stock screen:

a)    You can perform the following functions:

Monitor Stock

View the stock range

Return the stock range

Transfer the stock range

Allocate stock

View the blacklisted stocks

b)    Click Close.

The List Stock Holder screen reappears.

Field/Button Description – Monitor Stock screen

 

6.      To exit the screen, click Close.

Field/Button Description – List Stock Holder screen

Delete a stock holder

 

1.      Select the Airline Functions tab. On the Stock Control menu, go to Stock Holder and click List to get the List Stock Holder screen.

 

2.      In the List Stock Holder screen, list the details of a stock holder for a stock holder type. (mandatory)

 

3.      In the Stock Holder Details table, select a row. (mandatory)

 

4.      Click Delete. (mandatory)

Click Yes in the confirmation message that appears. (mandatory)

The stock holder is deleted.

 

5.      To exit the screen, click Close.
