STOCK CONTROL

Allocate New Stock
(Screen: STK001)

 


To access the Allocate New Stock screen:
Select the Airline Functions tab. On the Stock Control menu, go to Stock and click Allocate New.

 

The Allocate Stock screen can be used by the Cargo Headquarters’ Staff, Stock Control Manager, Regional Cargo Manager and Cargo Station Manager to allocate stock without any requests made by stock holders of a lower level

 

The values for the document type and its subtype will be displayed on the Doc Type and Sub Type respectively by default and these values can be changed if required

Functions

       Allocate stock to a stock holder without a request being raised

 

 

       Related Topics

Allocate stock to a stockholder against a stockholder request

 

 


       Allocate stock t o a stock holder without a request being raised

1.        Select the Airline Functions tab. On the Stock Control menu, go to Stock and click Allocate New to get the Allocate New Stock screen.

 

2.        Specify the search criteria by which the available stock can be searched and listed. The following fields are mandatory.

 

-           Doc Type

-           Sub Type

-           Approver

-           Stock Holder Code

-           Range From

 

3.        Click List.

The matching stock ranges for the criteria specified are displayed in the Available Stock section. The total number of documents available for allocation is displayed in the Total no of Docs field.

 

4.        Select the range to be allocated and click >> to move it to the table on the right.

 

5.        In this table, you can modify the Range From and Range To values of the desired stock range to be allocated.

Note:
-     Click the Add link to add different ranges of stock.
-     The total number of stock that has been allocated is displayed in the Total no of Docs field.
-     Only the stock in the available stock range can be allocated.

 

6.        Click Save.

 

7.        Click Yes in the confirmation message that appears.

The specified range is allocated to the specified stockholder.

Allocate New Stock Screen – Description of Screen Elements       

 


To access the Allocate New Stock screen:

Select the Airline Functions tab. On the Stock Control menu, go to Stock and click Allocate New.

Function Description – Allocate New Stock screen

 

Description of Fields and Buttons

Fields and buttons are described separately under different headings. They are given in the same order as they appear on screen.

Field Description

Button Description

 

In field descriptions, fields may be marked as Mandatory, Optional or Display Field.

Mandatory Field - Fields in which data entry is mandatory

Optional Field - Fields in which data entry is optional

Display Field - Non-editable fields in which data appears automatically

 

FIELD DESCRIPTION

 

Doc Type (Mandatory)

The type of the document whose details are to be searched and displayed

The default value AWB will be displayed in the field and this can be changed if required

Select this from the drop-down list

 

Sub Type (Mandatory)

The sub type of the document whose details are to be searched and displayed

The default value S will be displayed in the field and this can be changed if required.

Select this from the drop-down list

Note:
The values available in this drop-down list would depend upon the Doc Type selected

 

Manual check box (Optional)

Select this check box to search for the available printed stocks

 

Approver (Mandatory)

The approver of the stockholder to whom the stock is being allocated

 

Stock Holder Type (Optional)

The type of stockholder to whom the stock is being allocated

This can be Region, Station or Agent

Select this from the drop-down list

 

Stock Holder Code (Mandatory)

The stockholder to whom the stock is being allocated

Specify this in the field or select from the LOV

Note:
The values displayed in this LOV depend upon the Stock Holder Type selected. If no Stock Holder Type is selected, then all the available values are shown.

 

       AVAILABLE STOCK SECTION

 

Range From (Mandatory)

The number from which the document ranges are to be searched and displayed

 

Range To (Mandatory)

The number till which the document ranges are to be searched and displayed

 

No of Docs (Optional)

The number of available stocks that are to be searched and displayed, starting with the number specified in the Range From field

 

Note:
It is mandatory to specify:
-        Either Range From and Range To
-        or the Range From and No of Docs

 

       THE TABLE (LEFT)

 

Range From (Display Field)

The starting number of the range

 

Range To (Display Field)

The ending number of the range

 

No of Docs (Display Field)

The number of available documents in the range

 

Total no of Docs (Display Field)

The total number available stocks

 

       THE TABLE (RIGHT)

 

Range From (Optional)

The starting number of the document range to be allocated

Specify this in the field

 

Range To (Optional)

The ending number of the document range to be allocated

Specify this in the field

 

No of Docs (Display Field)

The number of stocks in the range displayed to be allocated

 

Total no of Docs (Display Field)

The total number of stocks allocated

 

BUTTON/LINK DESCRIPTION

 

List

Specify search criteria and click this button to get a range of stocks matching the search criteria.

 

>> 

Select the required range of documents from the ranges displayed in the Table (left) and click this button to move the selected range to the Table (right)

 

Add|Delete link

Click the Add link to add a new row in the table. Select a row and click the Delete link to delete the range.

 

Save

Click this button to save the entries made.

 

Close

Click this button to exit the screen.
