STOCK CONTROL

Monitor Stock
(Screen: STK007)

 


To access the Monitor Stock screen:

Select the Airline Functions tab. On the Stock Control menu, go to Stock and click Monitor

 

The Monitor Stock screen can be used by Stock controller, Regional Cargo Manager, Cargo Station Manager and General Sales Agent to monitor the stock at different levels (Head Quarters, Region, Station and Agent). The Headquarters value will be displayed in the screen by default and this can be changed is if required

 

 

Stock can be monitored at a level equal to or lower than the stock holder position in the hierarchy. You can view the range details of the available and allocated stocks and also return or transfer the available and allocated stock range.

 

The screen provides link to Create Stock, Allocate New Stock and Maintain Stock Request screens.

Monitor Stock

1.        Select the Airline Functions tab .On the Stock Control menu, go to Stock and click Monitor to get the Monitor Stock screen.

 

2.        In the Monitor Stock screen, specify the required search criteria. The following are mandatory:

-           Stock Holder Code

-           Document Type

-           Sub Type

 

3.        Click List.

The details that satisfy the specified search criteria appear in thetable.

 

Note :
Click Clear to clear all entries simultaneously.

4.        Click on the drop-down arrow in the displayed row. The details of the stock holders, for whom the displayed stock holder is the approver, are listed beneath that row.

 

5.        Select one row and click Monitor. The details corresponding to that stock holder code are displayed in a row. Click the drop-down arrow in the displayed row. The details of the stock holders, for whom the displayed stock holder is the approver, are listed beneath the selected row

 

6.        Click Close to exit the Monitor Stock screen.

Return the Stock

2.        In the Monitor Stock screen, specify the required search criteria. The following are mandatory:

-           Stock Holder Code

4.        C lick on the drop-down arrow in the displayed row. Then a level lower than the stock holder code in the hierarchy, for which the specified stock holder is the approver, appears beneath that row.

 

4.        Select one row and click Return.

The Return Stock Range screen appears

 

5.        Specify the Range From and Range To in the Range To Be Returned section.

 

6.        Click Return.

The specified Stock range is returned to the approver of that corresponding stock holder.

The HQ cannot return the stock.

 

7.        Click Close to exit the Return Stock Range screen

Transfer the Stock

You can transfer the available stock between the stock holders of the same level.

 

1.        Select the Airline Functions tab .On the Stock Control menu, go to Stock and click Monitor to get the Monitor Stock screen.

 

2.        In the Monitor Stock screen, specify the required search criteria. The following are mandatory:

-           Stock Holder Code

-           Document Type

-           Sub Type

 

3.        Click List.

The details that satisfy the specified search criteria appear in the table.

 

4.        Click on the drop-down arrow in the displayed row. Then a level lower than the stock holder code in the hierarchy, for which the specified stock holder is the approver, appears beneath that row.

 

5.        Select one row and click Transfer.

The Transfer Stock Range screen appears.

 

6.        Specify the value in the Transfer To field or select it from the LOV.

Note :
Transfer of stocks can be done only among stock holders at the same level

 

7.        If you want to transfer the stock by number of documents, then

Select the radio button By Number of Documents

-           Specify the Range From and Number of Documents fields.

OR

If you want to transfer the stock By Range, then

-           Select the radio button By Range

-           Specify the Range From and Range To in the Range To Be Transferred section

 

8.        Click Transfer.

The specified stock range is transferred.

 

9.   Click Close to exit the Transfer Stock Range screen

 

Field/Button Description – Transfer Stock Range screen

 

Note :
-     Click Clear to clear all the entries simultaneously

-           Click Close to exit the screen

Monitor Stock – Description of Screen Elements        

 


To access the Monitor Stock screen:
Select the Airline Functions tab. On the Stock Control menu, go to Stock and click Monitor.

Function Description – Monitor Stock screen

 

Description of Fields and Buttons

Fields and buttons are described separately under different headings. They are given in the same order as they appear on screen.

Field Description

Button Description

In field descriptions, fields may be marked as Mandatory, Optional or Display Field.

 

Mandatory - Fields in which data entry is mandatory

Optional - Fields in which data entry is optional

Display Field - Non-editable fields in which data appears automatically

 

FIELD DESCRIPTION

 

SEARCH CRITERIA SECTION

 

Stock Holder Type (Mandatory)

The type of the stock holder whose stock has to be monitored. It can be – Headquarters, Region, Station or Agent

Select this from the drop-down list.

Note:
The Headquarters value will be displayed in the screen by default and this can be changed is if required

 

Stock Holder Code (Mandatory)

The code of the stock holder whose stock has to be monitored.

Specify this in the field or select it from the LOV.

Note:
The values in the Stock Holder Code LOV depends on the Stock Holder Type selected.

 

Doc Type (Mandatory)

The Document Type of the stock that has to be monitored.

Select this from the drop-down list.

 

Sub Type (Mandatory)

The Sub Type of the stock that has to be monitored.

Select this from the drop-down list

Note:
The values in the Sub Type drop down depend on the Doc Type selected.

 

Manual check box (Optional)

Select this check box to monitor the printed stocks.

 

THE TABLE SECTION

 

Stock Holder (Display)

The code of the stock holder whose stock is monitored

 

Sub Type (Display)

The Sub Type of the stock, which is monitored.

 

Available Stock (Display)

The stock that is available with the stock holder

 

Allocated Stock (Display)

The stock that is allocated to the stock holder

 

Requests

Received (Display)

The number of stock requests received by the stock holder

 

Placed (Display)

The number of stock requests placed by the stock holder.

 

BUTTON DESCRIPTION

 

List

Specify the required search criteria details and click List. The matching stock holder details whose stock has to be monitored are populated in the table.

 

Clear

Click this button to clear all entries simultaneously

 

Monitor

Select one row from the table, and click Monitor to monitor the stock corresponding to the selected row.

 

View Range

Select one row from the table, and click View Range . The system would take you to the View Stock Range screen, where the user can view the details of the stock range.

 

Return

Selecting one row from the table, and clicking Return would take you to the Return Stock Range screen; the user can return the stock to the approver.

 

Transfer

On selecting one row from the table, and clicking Transfer would take you to the Transfer Stock Range screen, the user can transfer the stock to the stock holder at the same level.

 

Create Stock

Click Create Stock to navigate to the Create Stock screen, where new stock can be created.

 

Allocate Stock

Click Allocate Stock to navigate to the Create Stock screen, where stock can be allocated

 

Create Request

Click Create Request to go the Create Stock Request screen where you can create the stock request.

 

Close

Click Close to exit the screen
