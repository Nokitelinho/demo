STOCK CONTROL

BlackList Stock
(Screen: STK003)

To access the BlackList Stock screen:
Select the Airline Functions tab. On the Stock Control menu, go to Stock and click Blacklist.

 

The BlackList Stock screen can be used by the Cargo Headquarters’ Staff, Stock Control Manager, Regional Cargo Manager and Cargo Station Manager to blacklist stock within a specified range (document numbers).

       Blacklist a range of stock

1.        Select the Airline Functions tab. On the Stock Control menu, go to Stock and click Blacklist.

 

2.        In the BlackList Stock screen, specify the details of the stock to be blacklisted. The following details are mandatory:

-           Doc Type

-           Sub Type

-           Range From

-           Range To

 

3.        Specify Remarks, if any, regarding the blacklisted stock.

 

4.        Click Blacklist.

 

5.        Click Yes in the confirmation message that appears.

The list of stocks that lie within the range specified in the Range From and Range To fields get blacklisted and notifications are sent to whoever concerned.

 

6.        Click OK in the dialog box that appears to confirm the blacklisting.

Function Description – BlackList Stock screen

 

Description of Fields and Buttons

Fields and buttons are described separately under different headings. They are given in the same order as they appear on screen.

Field Description

Button Description

 

In field descriptions, fields may be marked as Mandatory, Optional or Display Field.

 

Mandatory Field - Fields in which data entry is mandatory

Optional Field - Fields in which data entry is optional

Display Field - Non-editable fields in which data appears automatically

 

FIELD DESCRIPTION

 

Doc Type (Mandatory)

The type of the document to be blacklisted

Select this from the drop-down list.

 

Sub Type (Mandatory)

The sub type of the document to be blacklisted

Select this from the drop-down list.

 

Note:
The Sub Type corresponding to the selected Doc Type appears in the field. If more than one Sub Type exists for a Doc Type, then select the required type from the drop-down list.

 

Stock Holder Code (Optional)

The code of the stock holder whose stock range is to be blacklisted

Specify this in the field or select from the LOV.

 

Range From - Range To (Mandatory)

The range (document numbers) of the stock to be blacklisted

Specify the ‘From’ document number and the ‘To’ document number in these fields to search for the list of blacklisted stocks that lies within that range.

 

Remarks (Optional)

Remarks on the blacklisted stock, if any

Specify this in the field.

 

BUTTON DESCRIPTION

 

Blacklist

Specify the required details and click this button to blacklist the specified stock.

 

Close

Click this button to exit the screen.
