STOCK CONTROL

Return Stock Range

 


To access the Return Stock Range screen:
-        Select the Airline Functions tab .On the Stock Control menu, go to Stock and click Monitor to get the Monitor Stock screen.
-        In this screen, specify Stock Holder Code , Doc Type and Sub Type and click List.
-        C lick the red drop-down arrow in the row. The list of stockholders appears.
-        Select a row from the table and click Return .

 

The Return Stock Range screen can be used by the Stock Holder to the return stocks from the available stock range to the approver for the stock holder.

 

This screen has two sections, Available Stock Range and Range To Be Returned. The Available Stock Range section is non-editable whereas the Range to be returned section is editable. You can specify the range to be returned and click Return to return stocks to the approver.

 

In field descriptions, fields may be marked as Mandatory, Optional or Display Field.

 

Mandatory - Fields in which data entry is mandatory

Optional - Fields in which data entry is optional

Display Field - Non-editable fields in which data appears automatically

 

FIELD DESCRIPTION

 

Stock Holder (Display Field)

The code of the stockholder, as specified in the Monitor Stock screen

 

Manual (Display Field)

This check box appears selected if the stock consists of printed documents (manual/hard copy

documents ). It appears deselected if the stock is system-generated.

 

Doc Type (Display Field)

The Document Type of the stock as specified in the Monitor Stock screen.

 

Sub Type (Display Field)

The Sub Type of the stock as specified in the Monitor Stock screen.

 

AVAILABLE STOCK RANGE SECTION

 

Range From - Range To (Display Field)

The range of serial numbers of the stock

The ‘From’ serial number and the ‘To’ serial number of the available stock appear in these fields.

 

No. Of Docs (Display Field)

The number of documents present within the range

 

Total Available Stock (Display Field)

The total number of documents available

This is the sum of all the numbers in the No. of Docs column.

 

RANGE TO BE RETURNED SECTION

 

Range From - Range To (Mandatory)

The range of serial numbers of the stock to be returned to the approver (of the specified stockholder)

Specify the ‘From’ serial number and the ‘To’ serial number in these fields.

Note :
-      This range should lie within the available stock range.
-      The Headquarters (HQ) cannot return the stock.

 

No. Of Docs (Display Field)

The number of documents present within the specified range

This automatically appears based on the range specified.

 

Total no. of Docs (Display Field)

The total number of documents to be returned to the approver (of the specified stockholder)

This is the sum of all the numbers in the No. of Docs column. This is automatically updated.

 

Remarks (Optional)

Additional remarks, if any

Specify this in the field.

 

BUTTONS/LINKS DESCRIPTION

 

Add|Delete links

Click the Add link to add a new row. Select a row and Click the Delete link to delete the row.

 

Return

Click this button to return the stock to the approver of the stockholder.

 

Close

Click this button to exit the screen.
