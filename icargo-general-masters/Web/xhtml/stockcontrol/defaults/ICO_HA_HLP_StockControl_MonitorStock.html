STOCK CONTROL

MONITOR STOCK

(Screen ID – STK007)

Click here to view screen access information.

To access the Monitor Stock screen:
Select the Airline Functions tab. On the Stock Control menu, go to Stock and click Monitor.

 

When you access the screen, default values are displayed in the following fields:

·        Stock Holder Type – ‘Headquarters’

·        Doc Type – ‘ AWB’

·        Sub Type – ‘S’

 

The screen provides links to Create Stock, Allocate New Stock and Maintain Stock Request screens.

 

The Stock Control Department can use the Monitor Stock screen.

Monitor stock

 

1.      Select the Airline Functions tab. On the Stock Control menu, go to Stock and click Monitor to get the Monitor Stock screen.

 

2.      In the Monitor Stock screen, specify the search criteria. The following are mandatory:

·        Select the Stock Holder Type.

·        Type or select the Stock Holder Code

·        Select the Doc Type

·        Select the SubType

 

3.      Click List. (mandatory)

The details that satisfy the specified search criteria appear in the Stock Holder Details and Distribution Details table.

 

4.      In the Distribution Details table, select the stock holder whose details are to be monitored. (mandatory).

 

5.      Click Monitor. (mandatory)

The details corresponding to that stock holder will be displayed in the Stock Holder Details table.

Note:

•     The stock details of all the levels for which the selected stockholder is the approver appear in the Distribution Details table.

For e.g.: If the stockholder is region, then the system displays the stock details of all the stations and agents below the respective region.

•     The Manual checkbox, if selected, displays the printed stocks in the Stock Holder Details and Distributions Details section.

 

6.      To exit the screen, click Close.

View the stock range

 

1.      Select the Airline Functions tab. On the Stock Control menu, go to Stock and click Monitor to get the Monitor Stock screen.

 

2.      In the Monitor Stock screen, list the stock holders. (mandatory)

 

3.      In the Distribution Details table, select the stock holder whose stock range is to be viewed. (mandatory)

 

4.      Click View Range. (mandatory)

The View Range screen appears.

 

5.      In the View Range screen :

a)    The following details are displayed:

-     Available Stock Range

-     Total Available Stock

-     Allocated Stock Range

-     Total Allocated Stock

 

b)    Click Close

The Monitor Stock screen reappears.

Return the stock range

 

1.      Select the Airline Functions tab. On the Stock Control menu, go to Stock and click Monitor to get the Monitor Stock screen.

 

2.      In the Monitor Stock screen, list the stock holders. (mandatory)

 

3.      In the Distribution Details table, select the stock holder whose stock range is to be returned (mandatory).

 

4.      Click Return. (mandatory)

The Return Stock Range screen appears with the available stock range of the stock holder in the Available Stock Range table.

 

5.      In the Return Stock Range screen:

a)    In the Range To Be Returned table, type the Range From and the Range To of the stock to be returned. (mandatory)

Note:

To discontinue the process and return to the Monitor Stock screen, click Close

 

b)    Click Return. (mandatory)

Click Yes in the confirmation message that appears.

The stocks in the specified range are returned to the approver of that stock holder.

The Monitor Stock screen reappears.

Note:

·          The stocks of headquarters cannot be returned.

·          Stocks returned will go back to the approver of the stockholder.

Field/Button Description – Return Stock Range screen

 

6.      To exit the screen, click Close.

Transfer the stock range

 

1.      Select the Airline Functions tab. On the Stock Control menu, go to Stock and click Monitor to get the Monitor Stock screen

 

2.      In the Monitor Stock screen, list the stock holders. (mandatory)

 

3.      In the Distribution Details table, select the stock holder whose stock range is to be transferred (mandatory).

 

4.      Click Transfer. (mandatory)

The Transfer Stock Range screen appears with the available stock range of the stockholder in the Available Stock Range section.

 

5.      In the Transfer Stock Range screen:

a)    Type or select the stockholder to whom the stock is to be transferred in the Transfer To and type the reason in the Reason field. (mandatory)

Note:

•     The stocks of the Headquarters cannot be transferred.

•     Transfer of stocks can be done only among stock holders at the same level.

 

b)     Stocks can be transferred by selecting either By Number of Documents or By Range. (optional)

•     If you have opted for By Number of Documents, type the Range From and Number of Documents

OR

•     If you have selected By Range, type the range to be transferred in the Range From and Range To fields in the Range To Be Transferred table.

The total number of documents will be automatically populated in the Total no of Docs field .

Note:

To discontinue the process and return to Monitor Stock screen, click Close

 

c)      Click Transfer. (mandatory)

Click Yes in the confirmation message that appears.

The stocks in the specified range are transferred.

The Monitor Stock screen reappears.

Field/Button Description – Transfer Stock Range screen

 

6.      To exit the screen, click Close.
