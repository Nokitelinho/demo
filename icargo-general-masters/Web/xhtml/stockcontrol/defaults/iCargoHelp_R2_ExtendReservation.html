Extend Reservation Screen    

To access the Extend Reservation screen:
-           Select the Cargo Terminal Opr tab.
-           On the Export menu, go to AWB andclick List AWB Reservation.
-           In the Reservation Listing screen that appears, give the search criteria and click List.
-           From the list, s elect the reservations to be extended.
-           Click Extend Reservation.

 

In the Reservation Listing screen, i f more than one reservation has been selected for extension, the AWB No., current Expiry Date and Remarks of the first selected reservation is displayed in the Extend Reservation screen. 

 

1.        Change the Expiry Date of the displayed reservation.

Note :

The expiry date can be changed to a future date.

2.        Use the First| Next| Previous| Last links to go to the remaining reservations. Now change corresponding expiry dates.

3.        Click OK.

 

 

In field descriptions, fields may be marked as Mandatory, Optional or Display Field.

 

Mandatory - Fields in which data entry is mandatory

Optional - Fields in which data entry is optional

Display Field - Non-editable fields in which data appears automatically

 

FIELD DESCRIPTION

 

AWB (Display Field)

The unique bill number on which a shipment travels

 

Expiry Date (Optional Field)

The date on which the reservation of the AWB expires

Specify the new date or select from the calendar.

 

Remarks (Optional Field)

The remark specified against the reservation in the Reservation Listing screen appears here.

This can be modified if required.

 

BUTTON/LINK DESCRIPTION

 

First|Next|Previous|Last links

Click these links to move to the first/next/previous/last page, in the table.

 

OK

Click this button to save all the extended reservations and return to Reservation Listing screen.

 

Cancel

Click this button to cancel the details specified and return to the Reservation Listing screen.
