Reservation
(Screen: STK008)

To access the Reservation screen:
Select the Cargo Terminal Opr tab. On the Export menu, go to AWB and click AWB Reservation .

 

The Reservation screen is used by a Ground Handling Agents to reserve AWBs in advance for his customers. The airline will provide the Ground Handling Agents with some AWBs . The agent, with whom the customers book the shipment, can distribute the AWBs to the customers.

Reserve a Stock

1.        Select the Cargo Terminal Opr tab. On the Export menu, go to AWB and click AWB Reservation to get the Reservation screen.

 

2.        In this screen, specify the following:

·          Airline, of which you have stock in hand

·          AWB Type . The AWB Type can be manual AWB or system AWB.

·          Cust Code

·          Expiry Date , after which the reservation of that particular AWB expires.

·          Remarks

 

3.        AWB’s can be reserved in three manners:

·          To reserve an AWB by selecting General:

Click General and specify the number of AWB’s that are needed to be reserved for that particular customer.

·          To reserve an AWB by selecting Specific:

Select Specific and click the Add link. A new row appears where the AWB prefix of that particular Airline is shown. Provide the Document number in the second column.

Note :
You can also reserve an AWB by selecting both General and Specific.

 

4.        Click Reserve to reserve the Stock.

The Reserved AWB’s screen appears with the AWB’s that have been reserved.

Description of Fields and Buttons

Fields and buttons are described separately under different headings. They are given in the same order as they appear on screen.

Field Description

Button/Link Description

[a1]  

In this description, fields are marked in the following manner:

 

Mandatory - Fields in which data entry is mandatory

Optional - Fields in which data entry is optional

Display Field - Non-editable fields in which data appears automatically

 

FIELD DESCRIPTION

 

Airline (Mandatory)

The Airline provides the Ground Handling agent with some stocks. The Ground Handling agent has to specify which Airline’s stock he is going to reserve for a particular customer.

Specify the Airline Carrier Code in the field.

 

AWB Type (Mandatory)

The AWB type can be either manual AWB’s or system AWB’s.

-            Manual AWB’s are those that the Ground Handling agent has with him in hand. i.e. a hard copy, of which he will have to enter all the necessary details.

-            System AWB’s are those wherein the system generates the AWB on the spot. The Ground Handling agent fills in the necessary details and then orders a print to be given to the shipper.

Select this from the drop drop- down list.

 

Cust . Code (Mandatory)

The code of a customer for whom the stock is to be booked

Specify this in the field or select from the LOV.

 

Expiry Date (Display Field)

The date after which the reservation of the stock becomes invalid

 

Remarks (Mandatory)

Remarks regarding the stock reservation

Specify this in the field.

 

GENERAL checkbox (Optional)

If the General checkbox is selected, then the specified no: number of AWB’s from the stock available for the specified Airline will be reserved

 

No of AWBs

The number of AWB’s to be reserved

Note :
This is mandatory if General stock is given for reservation

 

SPECIFIC checkbox (Optional)

If the Specific checkbox is selected, then the AWB’s specified by the GHA [a2]   will be reserved

 

THE TABLE

 

AWB

The AWB(s) to be reserved

Note :
-     This is mandatory if Specific stock is given for reservation
-     Either specific or general or both should be specified

 

BUTTON/LINK DESCRIPTION

 

Add | Delete links

Click the Add link to add a row under Specific section. Click the Delete link to delete a row of AWB, under Specific section.

 

Reserve

Specify the details and click this button to reserve the AWB(s).

 

Clear

Click this button to clear all entries simultaneously.

 

Close

Click this button to exit the screen.
