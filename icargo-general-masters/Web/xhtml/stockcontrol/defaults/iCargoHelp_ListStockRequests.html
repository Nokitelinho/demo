STOCK CONTROL

List Stock Requests
(Screen: STK016)

 


To access the List Stock Requests screen :
Select the Airline Functions tab. On the Stock Control menu, go to Stock Request and click List.

 

The List Stock Requests screen can be used by Stock Control Manager, Regional Cargo Manager, Cargo Station Manager or the General Sales Agents to view the stock requests made by different regions, stations or agents in the airline network. You can specify your search criteria to get a list of matching stock requests.

 

The current date at the logged in station is displayed in both the From Date and To Date fields.

This screen provides a link to the Maintain Stock Request screen.

View list of stock requests

1.        Select the Airline Functions tab. On the Stock Control menu, go to Stock Request and click List to get the List Stock Requests screen.

 

2.        In this screen, specify the required search criteria. The following are mandatory:

-           Doc Type

-           Sub Type

-           From Date

-           To Date

 

3.        Click List.

The list of stock requests that satisfy the specified search criteria appears in the Stock Request Details section.

Note :
Click Clear to clear all search criteria entries simultaneously.

 

4.        Click Close to exit the screen.

 

Field/Button Description – List Stock Requests screen

 

View/Modify details of a stock request

 

1.        Select the Airline Functions tab. On the Stock Control menu, go to Stock Request and click List to get the List Stock Requests screen.

 

2.        In this screen, specify the search criteria and click List.

A list of stock requests that match the search criteria appears in the Stock Request Details section.

 

3.        Select the stock request whose details you wish to view/modify.

 

4.        Click Details.

The Maintain Stock Request screen appears with details of the stock request.

 

5.        In this screen, modify the required stock request details.

Note :
Req. Ref No, Date of Req., Status, Allocated Stock and App/Rej. Remarks cannot be modified.

 

Field/Button Description - Maintain Stock Request screen

 

6.        Click Save in the Maintain Stock Request screen.

 

7.        Click Yes in the confirmation message that appears.

 

8.        Click Close to exit the Maintain Stock Request screen.

The updated stock request is displayed in the List Stock Requests screen

 

9.        Click Close to exit the screen.

 

Field/Button Description – List Stock Requests screen

 

Cancel a stock request

You can change the status of a stock request to ‘Cancelled’.

 

1.        Select the Airline Functions tab. On the Stock Control menu, go to Stock Request and click List to get the List Stock Requests screen.

 

2.        In this screen, specify the search criteria and click List.

A list of stock requests that match the search criteria appears in the Stock Request Details section.

 

3.        Select the stock request to be cancelled.

 

4.        Click Cancel.

 

5.        Click Yes in the confirmation message that appears.

The status of the selected stock request is changed to ‘Cancelled’.

 

6.        Click Close to exit the screen.

List Stock Requests- Description of Screen Elements

 


To access the List Stock Requests screen :
Select the Airline Functions tab. On the Stock Control menu, go to Stock Request and click List.

Function Description – List Stock Requests screen

 

Description of Fields and Buttons

Fields and buttons are described separately under different headings. They are given in the same order as they appear on screen.

Field Description

ButtonDescription

 

In field descriptions, fields may be marked in the following manner:

 

Mandatory - Fields in which data entry is mandatory

Optional - Fields in which data entry is optional

Display Field - Non-editable fields in which data appears automatically

 

FIELD DESCRIPTION

 

SEARCH CRITERIA SECTION

 

Req. Ref. No (Optional)

The request reference number uniquely identifies a stock request. It is automatically generated when a new request is created.

Specify this to search for a stock request with a particular Req. Ref. No.

 

Status (Optional)

The status of the stock requests – Approved, Cancelled, Allocated, Completed, New or Rejected

Select this from the drop-down list to search for requests with any particular status.

 

Doc Type (Mandatory)

The document type of the stock for which request is placed

Select this from the drop-down list.

 

Sub Type (Mandatory)

The sub type of the stock for which request is placed.

Select this from the drop-down list. The values in the drop-down list depend on the Doc Type selected.

 

Stock Holder Type (Optional)

The type of the stockholder who has raised the request – Headquarters, Region, Station or Agent

Select this from the drop-down list.

 

Stock Holder Code (Optional)

The code of the stockholder who has raised the request

Specify this in the field or select it from the LOV . The values in the Stock Holder Code LOV depend on the Stock Holder Type selected.

 

From Date , To Date (Mandatory)

You can specify a period and the system searches for all stock requests raised during that period.

The current date appears in both fields, by default.

Specify the From Date and To Date of the required period or select from the calendar.

 

Manual check box (Optional)

Select this check box to search for stock requests placed on the printed stocks.

 

 

 STOCK REQUEST DETAILS SECTION

 

Req. Ref. No. (Display Field)

The request reference number uniquely identifies a stock request. It is automatically generated when a new request is created.

 

Requested by (Display Field)

The code corresponding to the stockholder who made the request

 

Date of request (Display Field)

The date on which the request was made

 

Doc Type (Display Field)

The document type of the stock for which request was placed

 

Sub Type (Display Field)

The sub type of the stock for which request was placed

 

No. of Docs (Display Field)

The number of documents requested

 

Status (Display Field)

Status of the request made

This can be Approved, Cancelled, Allocated, Completed, New or Rejected.

 

BUTTON/LINK DESCRIPTION

 

List

Specify search criteria and click this button to get a list of stock requests matching the search criteria.

 

Clear

Click this button to clear all entries simultaneously.

 

First l Next l Prev l Last

Click these links to move to the first/next/previous/last page, in the Stock request details section

 

Create

Click this button to access the Maintain Stock Request screen.

 

Details

Select a stock request and click this button to access the Maintain Stock Request screen.

You can view/modify the details of the stock request here.

 

Cancel

Select a stock request and click this button to cancel a stock request.

 

Close

Click this button to exit the screen.
