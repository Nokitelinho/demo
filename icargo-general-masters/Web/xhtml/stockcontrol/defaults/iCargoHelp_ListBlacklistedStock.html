STOCK CONTROL

List BlackListed Stock
(Screen: STK005)

 


To access the List BlackListed Stock screen:
Select the Airline Functions tab . On the Stock Control menu, click List BlackListed Stock.

 

Once a stock is blacklisted, it becomes non-operational. The List BlackListed Stock screen can be used by the Cargo Headquarters, Stock Control Manager, Regional Cargo Manager and Cargo Station Manager to view the list of blacklisted stocks and also to revoke them. You can specify your search criteria to get a list of matching blacklisted stocks.

 

The current date at the logged in station is displayed in both the From Date and To Date fields.

These can be modified if required.

       View list of blacklisted stock

1.        Select the Airline Functions tab . On the Stock Control menu, click List BlackListed Stock, to get the List Blacklisted Stock screen.

 

2.        In this screen, specify the required search criteria. The following are mandatory:

-           Document Type

-           Sub Type

-           From Date

-           To Date

 

3.        Click List.

The list of blacklisted stocks that satisfy the specified search criteria appears in the Blacklisted Stock Details section.

 

Note :
Click Clear to clear all entries simultaneously.

 

4.        Click Close to exit the screen.

 

Field/Button Description – List Blacklisted Stock screen

 

       Revoke blacklisted stock

 

1.        Select the Airline Functions tab . On the Stock Control menu, click List BlackListed Stock, to get the List Blacklisted Stock screen.

 

2.        In this screen, specify the required search criteria and click List.

A list of blacklisted stocks that match the search criteria appears in the Blacklisted Stock Details section.

 

3.        Select the blacklisted stock that is to be revoked.

 

4.        Click Revoke.

The Revoke Blacklisted Stock screen appears with details of the blacklisted stock.

 

5.        In this screen, you can revoke either the entire blacklisted stock or a subset of it. To revoke a subset of the stock, change the RangeFrom and Range To and then click Revoke.

Note :
-     The specified range should lie within the range of the selected blacklisted stock.

-     The Doc Type, Sub Type and Remarks fields cannot be edited.

-     To. revoke a whole blacklisted stock, click Revoke (without making any changes)

 

Field/Button Description – Revoke Blacklisted Stock screen

 

6.        Click Yes in the confirmation message that appears.

The stock/subset of the stock is revoked and the List Blacklisted Stock screen appears. The updated range is displayed here.

 

7.        Click Close to exit the List Blacklisted Stock screen.

List Blacklisted Stock – Description of Screen Elements

 


To access the List Blacklisted Stock screen:
Select the Airline Functions tab. On the Stock menu, go to Stock Control and click List Blacklisted Stock.

Function Description – List Blacklisted Stock screen

 

Description of Fields and Buttons

Fields and buttons are described separately under different headings. They are given in the same order as they appear on screen.

Field Description

Button Description

 

In field descriptions, fields may be marked as Mandatory, Optional or Display Field.

 

Mandatory - Fields in which data entry is mandatory

Optional - Fields in which data entry is optional

Display Field - Non-editable fields in which data appears automatically

FIELD DESCRIPTION

 

       SEARCH CRITERIA SECTION

 

Doc Type (Mandatory)

The document type of the stock that is blacklisted

Select this from the drop-down list to search for stocks of a particular document type.

 

Sub Type (Mandatory)

The sub type of the stock that is blacklisted

Select this from the drop-down list to search for stocks of a particular sub type.

Note:
The values in the Sub Type drop-down list depend on the Doc Type selected.

 

From Date , To Date (Mandatory)

You can specify a period and the system searches for all stocks that were blacklisted during that period.

The current date appears in both fields, by default.

Specify the From Da te and To Date of the required period or select from the calendar.

 

RangeFrom, Range To (Optional)

The range serial numbers of the stock

Specify the ‘From’ serial number and the ‘To’ serial number of in these fields to search for the list of blacklisted stocks that lies within that range.

 

       BLACKLISTED STOCK DETAILS SECTION

 

Doc Type (Display Field)

The document type of the blacklisted stock

 

Sub Type (Display Field)

The sub type corresponding to the document type of the blacklisted stock

 

Black Listing Date (Display Field)

The date on which the stock was blacklisted

 

RangeFrom - Range To (Display Field)

The range of serial numbers of the available stock (i.e. the ‘From’ serial number and the ‘To’ serial number)

 

Remarks (Display Field)

The remarks on the stock, if any

 

BUTTON/LINK DESCRIPTION

 

List

Specify the search criteria and click this button to view the list of the embargoes.

 

Clear

Click this button to clear all entries simultaneously.

 

First l Next l Prev l Last

Click these links to move to the first/next/previous/last page, in the table

 

Revoke

Select a stock and click this button to revoke the blacklisted stock.

The Revoke Blacklisted Stock screen appears with details of the blacklisted stock.

 

Close

Click Close to exit the screen.
