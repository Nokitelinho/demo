STOCK CONTROL

Create Stock
(Screen: STK004)

 


To access the Create Stock screen:
Select the Airline  Functions tab. On the Stock Control menu, go to Stock and click Maintain.

 

The Create Stock screen can be used by the Cargo Headquarters to record stock within a specified range of serial numbers. The stock can be either be hard copy documents (also called manual documents) or system-generated.

Create stock

You can record stocks in the system using the Create Stock screen. 

 

1.        Select the Airline Functions tab. On the Stock Control menu, go to Stock and click Maintain, to get the Create Stock screen.

 

2.        In this screen, specify the details of the stock. The following are mandatory:

-           Doc Type

-           Sub Type

 

3.        Specify the range of stock (in serial numbers) to be recorded in the RangeFrom and Range To fields.

Note :
- The RangeFrom serial number should be less than the Range To serial number .
- The number of documents that fall within the specified range appears in the No Of Docs field.
-You can specify more serial number ranges, if required. Click the Add link to add a new row and specify the new range there. Select a row and click the Delete link to delete it.

 

4.        Click Save.

A new range of stock is created

 

5.        Click Close to exit the screen

Create manual stock

You can maintain manual stocks in the system.  For this:

 

1.        Select the Airline Functions tab. On the Stock Control menu, go to Stock and click Maintain, to get the Create Stock screen.

 

2.        In the Create Stock screen, specify the details of the stock. The following are mandatory:

-           Doc Type

-           Sub Type

 

3.        Select the Manual check box, to indicate that the stocks being recorded are hard copy documents (manual documents).

 

4.        Specify the range of stock (in serial numbers) to be recorded in the RangeFrom and Range To fields.

Note :
- The RangeFrom serial number should be less than the Range To serial number .
-The number of documents that fall within the specified range appears in the No Of Docs field .
- You can specify more serial number ranges, if required. Click the Add link to add a new row and specify the new range there. Select a row and click the Delete link to delete it.

 

5.        Click Save.

A new range of manual stock is created.

 

6.        Click Close to exit the screen

Create Stock – Description of Screen Elements

 


To access the Create Stock screen:
Select the Airline Functions tab. On the Stock Control menu, go to Stock and click Maintain

 

Function Description – Create Stock Screen

 

Description of Fields and Buttons

Fields and buttons are described separately under different headings. They are given in the same order as they appear on screen.

Field Description

Button Description

 

In field descriptions, fields may be marked as Mandatory, Optional or Display Field.

 

Mandatory - Fields in which data entry is mandatory

Optional - Fields in which data entry is optional

Display Field - Non-editable fields in which data appears automatically

 

FIELD DESCRIPTION

 

Doc Type (Mandatory)

The Document Type of the stock to be maintained

Select this from the drop-down list.

 

Sub Type (Mandatory)

The Sub Type of the stock to be maintained

Select this from the drop-down list

Note:
The values in the Sub Type drop-down list depend on the Doc Type selected.

 

Manual check box (Mandatory when creating manual stock)

Select this check box, to indicate that the stocks being recorded are hard copy documents (manual documents).

 

RANGE TO BE CREATED SECTION

 

RangeFrom (Mandatory) – Range To (Mandatory)

The range of serial numbers of the stock to be recorded

Specify the ‘From’ serial number and the ‘To’ serial number in these fields.

 

You can record more than one range. Click the Add link to add a new row and specify the new range here.

 

No Of Docs (Display)

The number of documents present within the specified range

This automatically appears based on the range specified.

 

Total No:Of Docs (Display)

The total number of documents to be created

This is the sum of all the numbers in the No Of Docs column. This is automatically updated.

 

BUTTON/LINK DESCRIPTION

 

Add l Delete links

Click the Add link to add a new row in the table. Select a row and click the Delete link to delete the row.

 

Save

Click this button to save the entries.

 

Close

Click this button to exit the screen.
