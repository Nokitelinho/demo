STOCK CONTROL

Maintain Stock Holder
(Screen: STK014)

 


To access the Maintain Stock Holder screen:
Select the Airline Functions tab. On the Stock Control menu, go to Stock Holder and click Maintain.

 

The Maintain Stock Holder screen is used by the Cargo Headquarters for the creation of stock holder in the system. Stock holders can be Headquarters, Agent, Station or Region.    

Functions

Create a stock holder

View/Modify the details of an existing stock holder

 

Related Topics

View list of stockholders

 

 


Create a stock holder

1.        Select the Airline Functions tab. On the Stock Control menu, go to Stock Holder and click Maintain.

 

2.        In the Maintain Stock Holder screen, specify general details of the stock holder. The following are mandatory:

 

-           Stock Holder Type

-           Stock Holder Code

-           Stock Holder Name

-           Control Privilege

 

Note:
Click Display Details to see if the specified stock holder already exists. If the stock holder already exists, the details are populated in the screen.

 

3.        Click Yes in the confirmation message that appears

 

4.        Enter the details of the stock holder in the table section. The following details are mandatory

 

-           Doc Type

-           Sub Type

-           Approver

 

Note:
In the hierarchy, the approver cannot be in the same level or lower level than the level of the stock holder being created

 

5.        Click Save.

 

6.        Click Yes in the confirmation message that appears

A new stock holder is created

 

7.        Click OK in the dialog box that appears which confirms the creation of a stock holder

View/Modify the details of an existing stock holder

 

1.        Select the Airline Functions tab. On the Stock Control menu, go to Stock Holder and click Maintain.

 

2.        In the Maintain Stock Holder screen, specify the Stock Holder Code

 

3.        Click Display Details

The details of the stock holder corresponding to the code entered is displayed

 

4.        Modify the required details. The following details are non-editable

 

-     Stock Holder Type

-     Stock Holder Code

 

5.        Click Save.

 

6.        Click Yes in the confirmation message that appears. The modified details are saved

TABLE SECTION

 

Doc Type | Sub Type (Mandatory)

The type of the document created and the sub type corresponding to the created document type.

Select the Doc Type from the drop-down list. The Sub Type is populated based on the Doc Type selected. In case of documents with more than one sub type, select the desired sub-type from the dropdown.

 

Approver (Mandatory)

The approver for the created stock holder

Specify this in the field or select from the LOV

Note: 
In the hierarchy, the approver cannot be in the same level or lower level than the level of the stock holder being created

 

Reorder Level (Optional)

The number of documents with the stock holder such that if the number reaches or falls  below the reorder level specified, a request for the same has to be raised

Specify this in the field

 

Reorder Qty (Optional)

The number of documents to be ordered in case of a reorder

Specify this in the field

 

Reorder Alert check box (Optional)

Select this check box if an alert has to be sent automatically to the stock holder when the stock with him reaches the reorder level or falls below it.

Select the check box

 

Auto Stock Request check box (Optional)

Select this check box if a request has to be raised automatically when the stock level becomes equal to or falls below the reorder level specified

Select the check box

 

Autoprocess Quantity (Optional)

The quantity of stocks that has to be allocated automatically to the stock holder, whenever a request is generated by him

This allocation is done directly without approval, depending upon the requested stock and the stock availability with the approving authority

Specify this in the field

 

Remarks (Optional)

Remarks, if any, on the created stock holder

Specify this in the field

 

BUTTON/ LINK DESCRIPTION

 

Display Details

Specify the stock holder code and click this button to view details of thecorresponding stock holder.

 

Add l Delete links

Click Add to add a new row in the table. Select a row and click Delete to delete the selected stock holder

 

Save

Click this button to save the entries.

 

Clear

Click this button to simultaneously clear all entries.

 

Close

Click this button to exit the Maintain Stock Holder screen.
