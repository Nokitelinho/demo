STOCK CONTROL

Revoke BlackListed Stock

To access the Revoke Blacklisted Stock screen:
-        Select the Airline Functions tab. On the Stock Control menu, click List Blacklisted Stock to get the List BlackListed Stock screen.
-        Specify the Doc Type, Sub Type, From Date and To Date and click List to get a list of blacklisted stocks.
-        Select a row and click Revoke .

 

The Revoke Blacklisted Stock screen can be used by the Cargo Headquarters’ Staff, Stock Control Manager, Regional Cargo Manager and Cargo Station Manager to revoke a range of blacklisted stocks.

The revoked stocks are returned to the HQ and not to the stockholder who owned it previously.

 

The ‘From’ and ‘To’ serial numbers of the range of blacklisted stocks appear in the Range From and Range To fields. If the whole range is not to be revoked, this range can be edited if required.

In field descriptions, fields may be marked as Mandatory, Optional or Display Field

 

Mandatory - Fields in which data entry is mandatory

Optional - Fields in which data entry is optional

Display Field - Non-editable fields in which data appears automatically

 

FIELD DESCRIPTION

 

Doc Type (Display Field)

The type of the blacklisted document to be revoked

 

Sub Type (Display Field)

The sub type of the blacklisted document to be revoked

 

Range From - Range To (Mandatory)

The range (document numbers) of the stock to be revoke

Specify the ‘From’ document number and the ‘To’ document number in these fields to search for the list of blacklisted stocks that lies within that range.     

 

Remarks (Display Field)

Remarks on the revoked stock, if any

 

BUTTON DESCRIPTION

 

Notify

Specify the range and click this button (through email) to indicate the concerned person that the specified blacklisted stock is revoked

 

Revoke

Specify the range and click this button to revoke the specified range of blacklisted stock.

 

Close

Click this button to exit the screen.
