Stock Manager
(Screen: STK011)

To access the Stock Manager screen:
Select the Cargo Terminal Opr tab. On the Export menu, go to AWB Stock andclick Stock Manager.

 

The Stock Manager screen is used by the Stock Controller to process the request for stock of other airlines, create stock on the basis of the request and manage the stock created.

Stock List in the Stock List screen

 

 


View the list of stock

1.        Select the Cargo Terminal Opr tab. On the Export menu, go to AWB Stock and click Stock Manager to get the Stock Manager screen .

 

2.        In thisscreen, specify the required search criteria. The following are mandatory:

-           Airline

-           Doc Type

-           Doc Sub Type

 

3.        Click List.
The list of stocks that satisfy the search criteria gets displayed.

Note :
Click Clear to clear all the entries simultaneously

 

4.        Click Close to exit the screen

Add stock

 

1.        Select the Cargo Terminal Opr tab. On the Export menu, go to AWB Stock and click Stock Manager to get the Stock Manager screen .

 

2.        In thisscreen, specify the required search criteria. The following are mandatory:

-           Airline

-           Doc Type

-           Doc Sub Type

 

3.        Click List.
If the search criteria is matching then the details get displayed in the table section otherwise, the system will ask for confirmation to create a new stock allocation. Click Yes in the displayed message. Now you can proceed by recording all the required details.

Note:
Click Clear to clear all the entries simultaneously.

 

4.        Click Add to add stock details in the Add Stock screen.

 

5.        Enter the details of the stock to be created. The following are mandatory:

-     Range From

-     Range To

Note:
The Doc Type, Doc Sub Type, are non-editable fields.

 

6.        Click Ok to save the details entered and return to the Stock Manager screen.

 

7.        Click Cancel to return to the Stock Manager screen without saving the details entered.

Field/Button Description – Add Stock screen

 

8.        Click Close to exit the screen.

 

Field/Button Description – Stock Manager Screen

 

Remove stock

 

1.        Select the Cargo Terminal Opr tab. On the Export menu, go to AWB Stock and click Stock Manager to get the Stock Manager screen .

 

2.        In thisscreen, specify the required search criteria. The following are mandatory:

-           Airline

-           Doc Type

-           Doc Sub Type

 

3.        Click List.
If the search criteria are matching then the details get displayed in the table section otherwise, the system will ask for confirmation to create a new stock allocation. Click Yes in the displayed message. Now you can proceed by recording all the required details.

 

Note:
Click Clear to clear all the entries simultaneously.

 

4.        Click Remove button to remove the selected row of the stock details in the Remove Stock screen.

 

5.        Enter the details of the stock to be removed. The following are mandatory:

-     Range From

-     Range To

Note:
In the Current Range Details section the Range From, RangeTo, Doc Type, Doc Sub Type, are non-editable fields.

 

6.        Click Remove to remove the details entered and return to the Stock Manager screen.

 

7.        Click Cancel to return to the Stock Manager screen without saving the details entered.

Field/Button Description – Remove Stock screen

 

8.        Click Close to exit the screen.

Stock Manager – Description of Screen Elements       

 


To access the Stock Manager screen:
Select the Cargo Terminal Opr tab. On the Export menu, go to AWB Stock and click Stock Manager

Function Description – Stock Manager screen

 

Description of Fields and Buttons

Fields and buttons are described separately under different headings. They are given in the same order as they appear on screen.

Field Description

Button Description

 

In field descriptions, fields may be marked as Mandatory, Optional or Display Field.

 

Mandatory - Fields in which data entry is mandatory

Optional - Fields in which data entry is optional

Display Field - Non-editable fields in which data appears automatically

FIELD DESCRIPTION

 

SEARCH CRITERIA SECTION

 

Airline (Mandatory)

The airline whose stock details have to be viewed

Specify this in the field

 

Doc Type (Mandatory)

Type of the Document whose stock is to be listed

Select this form the drop down list.

Note
Doc.sub type will be changed corresponding to the Doc Type

 

Doc Sub Type (Mandatory)

Document sub type whose stock is to be listed

Select this form the drop down list.

Note
Doc.sub type will be changed corresponding to the Doc Type

 

TABLE SECTION

 

Doc Type (Display Field)

Type of the Document whose stock is to be listed

 

Doc Sub Type (Display Field)

Document sub type whose stock is to be listed

 

Range From (Display Field)

Starting number of the range of the Document.

 

Range To (Display Field)

Ending number of the range of the Document.

 

Count (Display Field)

Total number of the document available in stock.

 

Summary Count (Display Field)

Total of all the Documents of that Airline which is in Stock

 

BUTTON DESCRIPTION

 

List

Specify the required fields in the search criteriaand click this button to get the list of stocks satisfying the search criteria.

 

Clear

Click this button to clear all entries simultaneously.

 

Add

Click this button to add stock details in the Add Stock screen .

 

Remove

Click this button to remove stock details in the Remove Stock screen.

 

Close

Click this button to exit the screen.
